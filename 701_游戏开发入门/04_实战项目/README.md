# 实战项目

## 2D小游戏示例

### 项目简介
这是一个使用Python和Pygame库实现的简单贪吃蛇游戏示例。

### 功能特性
- 经典贪吃蛇玩法
- 分数统计
- 游戏结束界面
- 重新开始功能

### 运行环境
- Python 3.x
- Pygame库

### 安装依赖
```bash
pip install pygame
```

### 运行游戏
```bash
python 2D小游戏示例.py
```

### 游戏操作
- 方向键：控制蛇的移动方向
- R键：重新开始游戏
- Q键：退出游戏

### 游戏规则
- 控制蛇吃到红色食物来得分
- 蛇撞到自己身体游戏结束
- 蛇可以穿墙（从屏幕一侧穿出另一侧进入）
