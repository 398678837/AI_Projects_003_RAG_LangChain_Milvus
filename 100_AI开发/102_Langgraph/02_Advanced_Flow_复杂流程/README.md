# LangGraph复杂流程

本示例展示了如何使用LangGraph创建一个更复杂的流程，包含多个节点和更复杂的流程设计。

## 功能说明

- 创建复杂的状态图
- 实现多个节点和条件边
- 支持研究和验证流程
- 适配Qwen等通用大模型
- 支持批量测试和交互式测试

## 代码结构

```python
# 1. 配置API密钥和大模型
# 2. 定义状态类型
# 3. 实现节点函数
# 4. 实现条件边函数
# 5. 创建复杂状态图
# 6. 测试图
```

## 配置说明

1. **API密钥配置**：
   - 在环境变量中设置 `API_KEY`
   - 或直接在代码中修改 `api_key` 变量

2. **大模型配置**：
   - 默认使用Qwen模型的API地址
   - 可以根据需要修改 `base_url`

## 运行示例

1. 安装依赖：
   ```bash
   pip install -r ../../requirements.txt
   ```

2. 运行示例：
   ```bash
   python demo.py
   ```

3. 交互模式：
   - 输入查询，按回车获取回答
   - 输入'quit'退出

## 学习要点

- **复杂流程设计**：如何设计和实现更复杂的流程
- **多节点协作**：如何让多个节点协同工作
- **条件路由**：如何根据状态进行条件路由
- **状态传递**：如何在节点之间传递和共享状态
