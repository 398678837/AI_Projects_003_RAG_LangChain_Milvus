# LangGraph实战应用

本示例展示了一个完整的LangGraph应用，实现了一个智能问答系统。

## 功能说明

- 实现意图识别
- 知识检索
- 回答生成和验证
- 适配Qwen等通用大模型
- 支持批量测试和交互式测试

## 代码结构

```python
# 1. 配置API密钥和大模型
# 2. 定义模拟知识库
# 3. 定义状态类型
# 4. 实现节点函数
# 5. 实现条件边函数
# 6. 创建实战应用图
# 7. 测试应用
```

## 配置说明

1. **API密钥配置**：
   - 在环境变量中设置 `API_KEY`
   - 或直接在代码中修改 `api_key` 变量

2. **大模型配置**：
   - 默认使用Qwen模型的API地址
   - 可以根据需要修改 `base_url`

## 运行示例

1. 安装依赖：
   ```bash
   pip install -r ../../requirements.txt
   ```

2. 运行示例：
   ```bash
   python demo.py
   ```

3. 交互模式：
   - 输入查询，按回车获取回答
   - 输入'quit'退出
