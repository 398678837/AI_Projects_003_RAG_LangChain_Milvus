# 向量生成

## 什么是向量生成？

向量生成是指将文本、图像、音频等非结构化数据转换为向量表示的过程。向量表示是一种高维数字数组，可以用于语义检索、文本分类、聚类分析等任务。

## 常用的Embedding模型

### 1. sentence-transformers
- 轻量级的Embedding模型
- 支持多种语言和任务
- 适合快速做Demo

### 2. BERT
- 强大的预训练语言模型
- 支持多种下游任务
- 适合大规模生产环境

### 3. GPT
- 强大的预训练语言模型
- 支持多种下游任务
- 适合大规模生产环境

## 向量生成的步骤

1. **加载预训练的Embedding模型**: 选择合适的Embedding模型，如sentence-transformers、BERT等
2. **输入文本**: 将需要转换的文本输入模型
3. **生成向量**: 模型将文本转换为向量表示
4. **输出向量**: 将生成的向量输出，用于后续任务

## 向量的应用

### 1. 语义检索
- 输入问题向量，找出语义最相似的文本
- 广泛应用于RAG系统、搜索引擎等领域

### 2. 文本分类
- 将向量输入分类模型，进行文本分类
- 广泛应用于情感分析、垃圾邮件检测等领域

### 3. 聚类分析
- 将向量输入聚类模型，进行聚类分析
- 广泛应用于主题建模、用户分群等领域

### 4. 推荐系统
- 将向量用于个性化推荐
- 广泛应用于电商、视频等领域

## 总结

向量生成是将非结构化数据转换为向量表示的过程，广泛应用于语义检索、文本分类、聚类分析等任务。本目录提供了向量生成的基本概念和技术，帮助您快速掌握向量生成的技术。
