# LangChain集成外部工具

本示例展示了如何使用LangChain集成外部工具和API，增强模型的能力。

## 功能说明

- 集成外部工具：天气查询和时间查询
- 实现工具调用逻辑
- 将工具结果整合到回答中
- 适配Qwen等通用大模型
- 支持批量问答和交互式问答

## 代码结构

```python
# 1. 配置API密钥和大模型
# 2. 定义外部工具函数
# 3. 创建Prompt模板
# 4. 创建LLM链
# 5. 实现工具调用逻辑
# 6. 测试链
```

## 配置说明

1. **API密钥配置**：
   - 在环境变量中设置 `API_KEY`
   - 或直接在代码中修改 `api_key` 变量

2. **大模型配置**：
   - 默认使用Qwen模型的API地址
   - 可以根据需要修改 `base_url`

## 运行示例

1. 安装依赖：
   ```bash
   pip install -r ../../requirements.txt
   ```

2. 运行示例：
   ```bash
   python demo.py
   ```

3. 交互模式：
   - 输入问题，按回车获取回答
   - 输入'quit'退出

## 学习要点

- **外部工具集成**：如何将外部API和工具集成到LangChain中
- **工具调用逻辑**：如何根据问题自动调用相应的工具
- **结果整合**：如何将工具结果整合到模型回答中
- **模块化设计**：将工具调用逻辑与模型推理分离

## 扩展建议

- 集成更多外部工具，如搜索引擎、数据库查询等
- 使用LangChain的Toolkit和Agent框架，实现更智能的工具调用
- 添加工具调用的错误处理机制
