# LangChain评估与优化

本示例展示了如何评估和优化LangChain应用，包括性能评估和Prompt优化。

## 功能说明

- 实现基本的评估框架
- 比较不同Prompt的性能
- 测量响应时间和质量指标
- 适配Qwen等通用大模型
- 提供详细的评估结果

## 代码结构

```python
# 1. 配置API密钥和大模型
# 2. 定义评估数据集
# 3. 创建不同的Prompt模板
# 4. 创建链
# 5. 实现评估函数
# 6. 测试评估
```

## 配置说明

1. **API密钥配置**：
   - 在环境变量中设置 `API_KEY`
   - 或直接在代码中修改 `api_key` 变量

2. **大模型配置**：
   - 默认使用Qwen模型的API地址
   - 可以根据需要修改 `base_url`

## 运行示例

1. 安装依赖：
   ```bash
   pip install -r ../../requirements.txt
   ```

2. 运行示例：
   ```bash
   python demo.py
   ```

## 学习要点

- **评估框架**：如何构建基本的评估框架
- **性能指标**：如何测量响应时间和质量指标
- **Prompt优化**：如何通过优化Prompt提高性能
- **结果分析**：如何分析和比较不同配置的性能
