# LangChain记忆管理

本示例展示了如何使用LangChain实现记忆管理，支持多轮对话和上下文理解。

## 功能说明

- 实现对话记忆管理
- 支持多轮对话
- 保持对话上下文
- 适配Qwen等通用大模型
- 支持批量测试和交互式对话

## 代码结构

```python
# 1. 配置API密钥和大模型
# 2. 定义记忆类
# 3. 创建Prompt模板
# 4. 创建链
# 5. 测试记忆管理
```

## 配置说明

1. **API密钥配置**：
   - 在环境变量中设置 `API_KEY`
   - 或直接在代码中修改 `api_key` 变量

2. **大模型配置**：
   - 默认使用Qwen模型的API地址
   - 可以根据需要修改 `base_url`

## 运行示例

1. 安装依赖：
   ```bash
   pip install -r ../../requirements.txt
   ```

2. 运行示例：
   ```bash
   python demo.py
   ```

3. 交互模式：
   - 输入对话，按回车获取回答
   - 输入'quit'退出

## 学习要点

- **记忆管理**：如何实现和管理对话记忆
- **上下文理解**：如何让模型理解对话上下文
- **记忆长度控制**：如何控制记忆的长度，避免过长
- **多轮对话**：如何实现多轮对话功能

## 扩展建议

- 使用LangChain的官方记忆组件，如ConversationBufferMemory
- 实现更复杂的记忆策略，如摘要记忆
- 集成外部存储，如Redis、数据库等
- 添加记忆检索功能，支持基于历史对话的检索
