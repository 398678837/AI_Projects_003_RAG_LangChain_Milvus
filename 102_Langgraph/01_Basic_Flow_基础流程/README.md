# LangGraph基础流程

本示例展示了如何使用LangGraph创建一个基础的流程，包含2个节点和条件边判断问题类型。

## 功能说明

- 创建基本的状态图
- 实现节点和条件边
- 根据问题类型选择不同的处理路径
- 适配Qwen等通用大模型
- 支持批量测试和交互式测试

## 代码结构

```python
# 1. 配置API密钥和大模型
# 2. 定义状态类型
# 3. 实现节点函数
# 4. 实现条件边函数
# 5. 创建状态图
# 6. 测试图
```

## 配置说明

1. **API密钥配置**：
   - 在环境变量中设置 `API_KEY`
   - 或直接在代码中修改 `api_key` 变量

2. **大模型配置**：
   - 默认使用Qwen模型的API地址
   - 可以根据需要修改 `base_url`

## 运行示例

1. 安装依赖：
   ```bash
   pip install -r ../../requirements.txt
   ```

2. 运行示例：
   ```bash
   python demo.py
   ```

3. 交互模式：
   - 输入问题，按回车获取回答
   - 输入'quit'退出

## 学习要点

- **状态图**：如何创建和配置StateGraph
- **节点**：如何定义和实现节点函数
- **条件边**：如何实现条件路由
- **状态管理**：如何定义和使用状态类型
