# LangGraph状态管理

本示例展示了如何使用LangGraph管理和持久化状态，支持多轮对话和上下文保持。

## 功能说明

- 实现状态管理和持久化
- 支持对话历史记录
- 维护上下文信息
- 适配Qwen等通用大模型
- 支持批量测试和交互式测试

## 代码结构

```python
# 1. 配置API密钥和大模型
# 2. 定义状态类型
# 3. 实现节点函数
# 4. 创建状态管理图
# 5. 实现状态加载和持久化
# 6. 测试状态管理
```

## 配置说明

1. **API密钥配置**：
   - 在环境变量中设置 `API_KEY`
   - 或直接在代码中修改 `api_key` 变量

2. **大模型配置**：
   - 默认使用Qwen模型的API地址
   - 可以根据需要修改 `base_url`

## 运行示例

1. 安装依赖：
   ```bash
   pip install -r ../../requirements.txt
   ```

2. 运行示例：
   ```bash
   python demo.py
   ```

3. 交互模式：
   - 输入消息，按回车获取回答
   - 输入'quit'退出

## 学习要点

- **状态定义**：如何定义和使用状态类型
- **状态传递**：如何在节点之间传递状态
- **状态持久化**：如何保存和加载状态
- **上下文管理**：如何维护和更新上下文信息
