# Keras模型基础

本示例展示了Keras的模型基础概念和操作，包括模型创建、编译、训练、评估、预测以及模型的保存和加载。

## 功能说明

- 模型创建：使用Sequential API和Functional API创建模型
- 模型编译：使用不同的优化器编译模型
- 模型训练：训练模型并查看训练历史
- 模型评估：评估模型在测试集上的性能
- 模型预测：使用训练好的模型进行预测
- 模型保存和加载：保存训练好的模型并加载使用

## 学习要点

- **模型创建**：使用Sequential API和Functional API创建不同类型的模型
- **模型编译**：配置优化器、损失函数和评估指标
- **模型训练**：设置训练参数，如 epochs、batch_size 和 validation_split
- **模型评估**：评估模型在测试集上的性能
- **模型预测**：使用训练好的模型进行预测
- **模型保存和加载**：保存训练好的模型并在需要时加载使用
