# Keras迁移学习

本示例展示了Keras的迁移学习的使用方法，包括加载预训练模型、冻结预训练模型层、构建迁移学习模型、数据预处理、训练迁移学习模型、微调和实际应用。

## 功能说明

- 加载预训练模型：加载MobileNetV2预训练模型
- 冻结预训练模型层：冻结预训练模型的层，只训练新添加的层
- 构建迁移学习模型：在预训练模型的基础上添加新的分类层
- 数据预处理：预处理CIFAR-10数据集以适应模型要求
- 训练迁移学习模型：训练新添加的层
- 微调模型：解冻部分预训练层进行微调
- 迁移学习的应用：使用训练好的模型进行预测
