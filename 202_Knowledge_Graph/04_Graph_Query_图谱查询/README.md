# 知识图谱查询

本示例展示了知识图谱的查询方法，包括基础查询操作、基于节点的查询、基于关系的查询、路径查询、模式匹配查询、属性过滤查询、复杂查询和子图查询等。

## 功能说明

- 构建示例知识图谱：创建一个包含人物和公司的知识图谱
- 基础查询操作：查询所有节点、边及其属性
- 基于节点的查询：查询特定节点的属性、邻居、入边和出边
- 基于关系的查询：查询特定关系的边和特定节点的特定关系
- 路径查询：查询两个节点之间的最短路径和所有简单路径
- 模式匹配查询：查询符合特定模式的边和关系
- 属性过滤查询：根据节点和边的属性进行过滤查询
- 复杂查询示例：查询Alice的朋友的雇主和在同一家公司工作的人
- 子图查询：创建和可视化公司相关的子图
- 知识图谱查询的应用：介绍知识图谱查询的主要应用场景
