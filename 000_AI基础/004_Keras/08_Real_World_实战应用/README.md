# Keras实战应用

本示例展示了Keras在实际应用中的使用，包括图像分类、文本分类、时间序列预测、模型部署准备和自定义回调函数。

## 功能说明

- 图像分类实战：使用卷积神经网络对CIFAR-10数据集进行分类
- 文本分类实战：使用全连接神经网络对IMDB电影评论进行情感分析
- 时间序列预测实战：使用LSTM网络预测时间序列数据
- 模型部署准备：将训练好的模型保存为H5格式和TFLite格式，为部署做准备
- 自定义回调函数：创建自定义回调函数，实现训练过程中的自定义逻辑
