# SciPy统计函数

本示例展示了SciPy的统计函数功能，包括描述性统计、概率分布、假设检验、相关分析等。

## 功能说明

- 描述性统计：计算数据的均值、中位数、标准差等统计量
- 概率分布：正态分布、二项分布、泊松分布等
- 假设检验：t检验、卡方检验等
- 相关分析：皮尔逊相关系数、斯皮尔曼相关系数、肯德尔相关系数
- 分布拟合：拟合数据到不同的概率分布
- 统计检验：正态性检验、方差齐性检验等
- 可视化：直方图、箱线图、散点图等
