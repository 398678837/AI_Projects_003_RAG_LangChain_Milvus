# SciPy优化

本示例展示了SciPy的优化功能，包括无约束优化、有约束优化、全局优化、最小二乘优化等。

## 功能说明

- 无约束优化：使用minimize进行无约束优化
- 有约束优化：使用minimize进行有约束优化
- 全局优化：使用differential_evolution进行全局优化
- 最小二乘优化：使用curve_fit进行曲线拟合
- 方程求解：使用fsolve求解方程
- 线性规划：使用linprog求解线性规划问题
- 二次规划：使用minimize求解二次规划问题
- 多目标优化：使用标量化方法进行多目标优化
