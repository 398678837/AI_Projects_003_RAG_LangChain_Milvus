# SciPy积分

本示例展示了SciPy的积分功能，包括一维积分、二重积分、三重积分、常微分方程求解等。

## 功能说明

- 一维积分：使用quad进行数值积分
- 二重积分：使用dblquad进行二重积分
- 三重积分：使用tplquad进行三重积分
- 固定步长积分：使用trapezoid和simpson方法进行积分
- 常微分方程（ODE）求解：使用solve_ivp求解常微分方程
- 偏微分方程（PDE）求解：使用有限差分法求解热传导方程
- 积分变换：使用傅里叶变换
