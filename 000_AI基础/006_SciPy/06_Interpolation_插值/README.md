# SciPy插值

本示例展示了SciPy的插值功能，包括一维插值、二维插值、样条插值、径向基函数插值等。

## 功能说明

- 一维插值：使用interp1d进行线性和三次样条插值
- 二维插值：使用interp2d进行二维插值
- 样条插值：使用CubicSpline和UnivariateSpline进行样条插值
- 径向基函数插值：使用Rbf进行径向基函数插值
- 插值误差分析：分析不同插值方法的误差
- 不规则数据插值：使用griddata处理不规则数据
