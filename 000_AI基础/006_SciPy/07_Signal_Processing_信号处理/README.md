# SciPy信号处理

本示例展示了SciPy的信号处理功能，包括信号生成、滤波、频谱分析、信号检测等。

## 功能说明

- 信号生成：生成正弦信号、组合信号和带噪声的信号
- 滤波：使用巴特沃斯滤波器进行低通滤波
- 频谱分析：使用FFT进行频谱分析
- 信号检测：检测信号中的脉冲
- 相关分析：计算信号之间的互相关
- 信号变换：使用小波变换
- 系统识别：估计系统频率响应
- 窗函数：应用不同的窗函数
- 调制解调：进行信号调制和解调
