# 人脸识别

本示例展示了人脸识别的基本方法，包括人脸检测、人脸识别、人脸特征点检测和人脸表情识别等。

## 功能说明

- 人脸检测：使用Haar级联分类器检测人脸和眼睛
- 人脸识别：使用LBPHFaceRecognizer进行人脸识别
- 人脸特征点检测：检测人脸的关键特征点
- 人脸表情识别：识别不同的面部表情
