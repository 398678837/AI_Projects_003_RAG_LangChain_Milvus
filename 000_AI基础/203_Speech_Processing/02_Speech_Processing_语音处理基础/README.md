# 语音处理基础

本示例展示了语音处理的基础操作，包括音频预处理、滤波、重采样、分帧、加窗、特征提取和特征可视化等。

## 功能说明

- 音频预处理：生成示例音频并读取音频文件
- 音频归一化：归一化音频振幅
- 音频滤波：应用低通和高通滤波器
- 重采样：将音频重采样到不同的采样率
- 分帧：将音频分割为帧
- 加窗：对帧应用窗函数
- 特征提取：提取MFCC特征
- 特征可视化：可视化MFCC特征
- 音频波形对比：对比原始和滤波后的音频波形
- 语音处理基础技术总结：总结语音处理的基础技术

## 学习要点

- **音频预处理**：学习如何处理和准备音频数据
- **滤波技术**：学习如何应用不同类型的滤波器
- **重采样**：学习如何改变音频的采样率
- **分帧和加窗**：学习如何将音频分割为帧并应用窗函数
- **特征提取**：学习如何提取MFCC等音频特征
- **特征可视化**：学习如何可视化音频特征
