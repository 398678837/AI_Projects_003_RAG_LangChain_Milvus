# CV基础概念与环境

## 功能说明

- 环境检查：检查OpenCV、NumPy和Matplotlib版本
- 图像读取与显示：读取和显示图像
- 图像基本操作：调整大小、裁剪、旋转
- 图像直方图：计算和显示灰度直方图
- 边缘检测：使用Canny边缘检测
- 轮廓检测：查找和绘制轮廓
- 图像滤波：添加噪声和高斯滤波
