# Scikit-learn特征工程

本示例展示了Scikit-learn中各种特征工程方法，包括特征提取、特征选择、特征变换、特征组合、特征编码和特征工程管道。

## 功能说明

- 特征提取：从文本数据中提取特征，如CountVectorizer和TfidfVectorizer
- 特征选择：选择对模型最有价值的特征，如SelectKBest、RFE和SelectFromModel
- 特征变换：对特征进行各种变换，如标准化、多项式特征和自定义变换
- 特征组合：组合多个特征创建新特征，如多项式特征
- 特征编码：将分类数据转换为数值表示，如OneHotEncoder、LabelEncoder和OrdinalEncoder
- 特征工程管道：将多个特征工程步骤组合成一个流程

## 学习要点

- **特征提取**：从原始数据中提取有用的特征，如文本数据的向量化
- **特征选择**：选择对模型最有价值的特征，减少噪声和冗余
- **特征变换**：对特征进行各种变换，如标准化、归一化和多项式特征
- **特征组合**：组合多个特征创建新特征，提高模型性能
- **特征编码**：将分类数据转换为数值表示，使模型能够处理
- **特征工程管道**：将多个特征工程步骤组合成一个流程，提高效率和可重复性
