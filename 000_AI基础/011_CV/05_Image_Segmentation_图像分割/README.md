# 图像分割

本示例展示了图像分割的基本方法，包括基于阈值的分割、基于颜色的分割、基于边缘的分割、基于区域的分割和基于聚类的分割等。

## 功能说明

- 基于阈值的分割：使用全局阈值和自适应阈值
- 基于颜色的分割：使用HSV颜色空间进行颜色分割
- 基于边缘的分割：使用Canny边缘检测和轮廓分析
- 基于区域的分割：使用分水岭算法
- 基于聚类的分割：使用K-means聚类
- 分割结果分析：计算分割区域的面积、周长和中心点
