# 图像分类

本示例展示了图像分类的基本方法，包括数据准备、模型训练、模型评估、特征提取和模型保存等。

## 功能说明

- 数据准备：创建形状分类数据集（圆形、正方形、三角形）
- 模型训练：使用MLP分类器进行模型训练
- 模型评估：计算准确率和混淆矩阵
- 特征提取：提取图像的均值、标准差和边缘密度特征
- 模型保存与加载：保存和加载训练好的模型
