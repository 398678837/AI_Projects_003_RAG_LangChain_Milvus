# 机器人学习

## 什么是机器人学习？

机器人学习是人工智能的一个分支，专注于使机器人能够从数据和经验中学习。它结合了机器学习、计算机视觉、机器人学和控制理论，创建能够执行复杂任务的智能机器人。
