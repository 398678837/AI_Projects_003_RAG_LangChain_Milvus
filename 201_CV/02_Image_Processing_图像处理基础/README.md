# 图像处理基础

本示例展示了图像处理的基本操作，包括颜色空间转换、阈值处理、形态学操作、图像变换、滤波和边缘检测等。

## 功能说明

- 颜色空间转换：BGR转RGB、灰度、HSV
- 图像阈值处理：全局阈值和自适应阈值
- 形态学操作：腐蚀、膨胀、开运算、闭运算等
- 图像变换：平移、旋转、缩放、仿射变换、透视变换
- 图像滤波：均值滤波、高斯滤波、中值滤波、双边滤波
- 边缘检测：Canny、Sobel、Laplacian边缘检测

## 学习要点

- **颜色空间**：了解不同颜色空间的特点和应用场景
- **阈值处理**：学习如何将灰度图像转换为二值图像
- **形态学操作**：学习如何处理图像的形状和结构
- **图像变换**：学习如何对图像进行几何变换
- **图像滤波**：学习如何去除图像噪声
- **边缘检测**：学习如何提取图像的边缘信息
