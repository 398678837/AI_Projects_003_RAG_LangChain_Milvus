# FastAPI响应处理

本示例展示了FastAPI的响应处理功能，包括不同类型的响应、自定义响应等。

## 功能说明

- JSON响应：返回JSON格式的数据
- 自定义JSON响应：自定义JSON响应的状态码和头信息
- HTML响应：返回HTML格式的内容
- 纯文本响应：返回纯文本格式的内容
- 重定向响应：重定向到其他URL
- 文件响应：返回文件内容
- 流式响应：返回流式数据
- 字节响应：返回字节数据
- 错误响应：返回错误信息
- 自定义错误响应：自定义错误响应的格式
- 响应模型：使用Pydantic模型定义响应格式
- 响应模型列表：返回符合响应模型的列表
- 响应状态码：设置响应的状态码
- 响应头：设置响应的头信息
- 空响应：返回空响应

## 配置说明

1. **依赖包**：
   - fastapi
   - uvicorn
   - pydantic

2. **安装依赖**：
   ```bash
   pip install -r ../../requirements.txt
   ```

## 运行示例

1. 运行示例代码：
   ```bash
   python demo.py
   ```
   或
   ```bash
   uvicorn demo:app --reload
   ```

2. 访问方式：
   - JSON响应：http://127.0.0.1:8000/json
   - 自定义JSON响应：http://127.0.0.1:8000/custom-json
   - HTML响应：http://127.0.0.1:8000/html
   - 纯文本响应：http://127.0.0.1:8000/text
   - 重定向响应：http://127.0.0.1:8000/redirect
   - 文件响应：http://127.0.0.1:8000/file
   - 流式响应：http://127.0.0.1:8000/stream
   - 错误响应：http://127.0.0.1:8000/error
   - 响应模型：http://127.0.0.1:8000/item
   - 响应模型列表：http://127.0.0.1:8000/items
