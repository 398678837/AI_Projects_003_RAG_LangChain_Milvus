# FastAPI请求处理

本示例展示了FastAPI的请求处理功能，包括查询参数、路径参数、请求体、表单数据等。

## 功能说明

- 查询参数处理：处理URL查询参数
- 路径参数处理：处理URL路径参数
- 请求体处理：处理JSON请求体
- 混合参数处理：混合使用路径参数、请求体和查询参数
- 表单数据处理：处理表单提交的数据
- 文件上传处理：处理文件上传
- 多文件上传处理：处理多个文件上传
- 复杂请求体处理：处理多个请求体对象
- 原始请求体处理：处理原始字节请求体
- 查询参数列表：处理多个相同名称的查询参数

## 配置说明

1. **依赖包**：
   - fastapi
   - uvicorn
   - pydantic

2. **安装依赖**：
   ```bash
   pip install -r ../../requirements.txt
   ```

## 运行示例

1. 运行示例代码：
   ```bash
   python demo.py
   ```
   或
   ```bash
   uvicorn demo:app --reload
   ```

2. 访问方式：
   - 查询参数：http://127.0.0.1:8000/items/?skip=0&limit=10&q=test
   - 路径参数：http://127.0.0.1:8000/items/1
   - 请求体：POST http://127.0.0.1:8000/items/ 带JSON请求体
   - 表单数据：POST http://127.0.0.1:8000/login/ 带表单数据
   - 文件上传：POST http://127.0.0.1:8000/uploadfile/ 带文件
   - 查询参数列表：http://127.0.0.1:8000/items/list/?ids=1&ids=2&ids=3
