# LangChain智能代理

本示例展示了如何使用LangChain创建智能代理，实现工具调用和决策能力。

## 功能说明

- 实现智能代理的基本框架
- 集成计算器和天气查询工具
- 支持工具调用和直接回答
- 适配Qwen等通用大模型
- 支持批量测试和交互式模式

## 代码结构

```python
# 1. 配置API密钥和大模型
# 2. 定义工具类
# 3. 创建代理Prompt模板
# 4. 创建代理链
# 5. 实现代理处理流程
# 6. 测试代理
```

## 配置说明

1. **API密钥配置**：
   - 在环境变量中设置 `API_KEY`
   - 或直接在代码中修改 `api_key` 变量

2. **大模型配置**：
   - 默认使用Qwen模型的API地址
   - 可以根据需要修改 `base_url`

## 运行示例

1. 安装依赖：
   ```bash
   pip install -r ../../requirements.txt
   ```

2. 运行示例：
   ```bash
   python demo.py
   ```

3. 交互模式：
   - 输入请求，按回车获取回答
   - 输入'quit'退出

## 学习要点

- **代理原理**：如何创建一个能够使用工具的智能代理
- **工具集成**：如何集成和调用外部工具
- **决策机制**：如何让代理决定何时使用工具
- **流程控制**：如何实现代理的循环处理流程

## 扩展建议

- 集成更多工具，如搜索引擎、数据库查询等
- 使用LangChain的官方Agent框架，如AgentExecutor
- 添加记忆功能，支持多轮对话
- 实现更复杂的工具调用逻辑和参数解析
