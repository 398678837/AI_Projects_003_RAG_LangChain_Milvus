# LangChain实战应用

本示例展示了如何使用LangChain构建一个完整的RAG（Retrieval-Augmented Generation）系统。

## 功能说明

- 实现完整的RAG系统流程
- 包含文档检索和生成回答两个核心步骤
- 使用模拟文档库进行演示
- 适配Qwen等通用大模型
- 支持批量问答和交互式问答

## 代码结构

```python
# 1. 配置API密钥和大模型
# 2. 定义模拟文档库
# 3. 实现检索函数
# 4. 创建RAG Prompt模板
# 5. 创建RAG链
# 6. 实现完整的RAG流程
# 7. 测试RAG系统
```

## 配置说明

1. **API密钥配置**：
   - 在环境变量中设置 `API_KEY`
   - 或直接在代码中修改 `api_key` 变量

2. **大模型配置**：
   - 默认使用Qwen模型的API地址
   - 可以根据需要修改 `base_url`

## 运行示例

1. 安装依赖：
   ```bash
   pip install -r ../../requirements.txt
   ```

2. 运行示例：
   ```bash
   python demo.py
   ```

3. 交互模式：
   - 输入问题，按回车获取回答
   - 输入'quit'退出

## 学习要点

- **RAG系统原理**：结合检索和生成的技术
- **文档检索**：如何从文档库中检索相关信息
- **Prompt设计**：如何设计有效的RAG提示模板
- **完整流程**：从问题输入到回答生成的完整流程

## 扩展建议

- 使用真实的文档库，如PDF、网页等
- 集成向量数据库，如Milvus、Pinecone等
- 优化检索算法，提高检索准确性
- 添加文档更新机制，保持知识库的时效性
