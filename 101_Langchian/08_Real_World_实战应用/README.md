# LangChain实战应用

本示例展示了一个完整的LangChain应用，实现了一个简单的客服系统。

## 功能说明

- 实现产品查询功能
- 构建客服对话系统
- 基于产品信息生成回答
- 适配Qwen等通用大模型
- 支持批量测试和交互式对话

## 代码结构

```python
# 1. 配置API密钥和大模型
# 2. 定义模拟产品数据库
# 3. 实现产品查询函数
# 4. 创建客服Prompt模板
# 5. 创建客服链
# 6. 实现客服处理流程
# 7. 测试客服系统
```

## 配置说明

1. **API密钥配置**：
   - 在环境变量中设置 `API_KEY`
   - 或直接在代码中修改 `api_key` 变量

2. **大模型配置**：
   - 默认使用Qwen模型的API地址
   - 可以根据需要修改 `base_url`

## 运行示例

1. 安装依赖：
   ```bash
   pip install -r ../../requirements.txt
   ```

2. 运行示例：
   ```bash
   python demo.py
   ```

3. 交互模式：
   - 输入查询，按回车获取回答
   - 输入'quit'退出

## 学习要点
