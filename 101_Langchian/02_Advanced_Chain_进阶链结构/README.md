# LangChain进阶链结构

本示例展示了如何使用LangChain创建更复杂的链结构，如SequentialChain。

## 功能说明

- 使用SequentialChain创建顺序链
- 包含两个子链：分析链和回答链
- 实现多步骤处理流程
- 适配Qwen等通用大模型
- 支持批量问答和交互式问答

## 代码结构

```python
# 1. 配置API密钥和大模型
# 2. 创建Prompt模板（分析和回答）
# 3. 创建子链（分析链和回答链）
# 4. 创建顺序链
# 5. 测试链
```

## 配置说明

1. **API密钥配置**：
   - 在环境变量中设置 `API_KEY`
   - 或直接在代码中修改 `api_key` 变量

2. **大模型配置**：
   - 默认使用Qwen模型的API地址
   - 可以根据需要修改 `base_url`

## 运行示例

1. 安装依赖：
   ```bash
   pip install -r ../../requirements.txt
   ```

2. 运行示例：
   ```bash
   python demo.py
   ```

3. 交互模式：
   - 输入问题，按回车获取回答
   - 输入'quit'退出

## 学习要点

- **SequentialChain**：用于创建顺序执行的链
- **多输入输出**：在链之间传递数据
- **模块化设计**：将复杂任务分解为多个子任务
- **流程控制**：控制链的执行顺序

## 扩展建议

- 添加更多子链，构建更复杂的流程
- 集成RouterChain，根据条件选择不同的链
- 实现并行链，提高处理效率
- 添加错误处理机制，提高系统稳定性
