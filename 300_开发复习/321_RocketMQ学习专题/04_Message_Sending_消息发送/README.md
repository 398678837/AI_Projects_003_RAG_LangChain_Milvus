# 04_Message_Sending_消息发送

## 学习目标
- 掌握消息发送方式
- 理解同步发送
- 理解异步发送
- 理解Oneway发送
- 掌握发送配置

## 关键要点
### 1. 发送方式
- 同步发送
- 异步发送
- Oneway发送

### 2. 同步发送
- 等待响应
- 可靠性高
- 有返回值
- 适用于重要消息

### 3. 异步发送
- 不等待响应
- 回调通知
- 性能高
- 适用于高吞吐

### 4. Oneway发送
- 不等待响应
- 无回调
- 性能最高
- 可靠性最低
- 适用于日志等

### 5. 消息发送配置
- 重试次数
- 超时时间
- 发送队列
- 发送延迟

### 6. 消息发送状态
- SEND_OK
- FLUSH_DISK_TIMEOUT
- FLUSH_SLAVE_TIMEOUT
- SLAVE_NOT_AVAILABLE
