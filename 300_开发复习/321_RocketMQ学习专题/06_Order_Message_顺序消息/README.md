## 学习目标
- 理解顺序消息概念
- 掌握分区有序
- 掌握全局有序
- 理解顺序实现原理

## 关键要点
### 1. 顺序消息概念
- 全局有序
- 分区有序
- FIFO
- 同一队列有序

### 2. 分区有序
- 同一Key发送到同一队列
- 队列内有序
- 多个队列并行
- 性能较好

### 3. 全局有序
- 只有一个队列
- 完全有序
- 性能较低
- 适用于特定场景

### 4. 实现原理
- 消息队列选择
- Hash取模
- 同一队列消费
- 顺序消费监听器

### 5. 顺序消费
- MessageListenerOrderly
- 顺序消费
- 本地锁
- 分布式锁
