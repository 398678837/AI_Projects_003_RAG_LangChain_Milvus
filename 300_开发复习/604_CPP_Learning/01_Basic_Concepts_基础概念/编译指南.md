# C++基础概念编译指南

## 编译运行方法

### 方法1：使用Makefile

```bash
# 编译
make

# 运行
make run

# 清理
make clean
```

### 方法2：使用编译脚本

```bash
# 赋予执行权限
chmod +x compile.sh

# 运行脚本
./compile.sh
```

### 方法3：手动编译

```bash
# 使用Homebrew安装的g++编译器
/opt/homebrew/bin/g++-15 main.cpp -o main -std=c++17 -Wall -Wextra -pedantic

# 运行
./main
```

## 常见问题

### 问题：fatal error: 'iostream' file not found

**解决方案**：
1. 安装Homebrew：`/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"`
2. 安装GCC：`brew install gcc`
3. 使用Homebrew安装的g++编译器编译代码

### 问题：command not found: g++-15

**解决方案**：
- 确保已安装GCC：`brew install gcc`
- 检查g++-15的路径：`find /opt/homebrew -name 'g++-15'`
- 使用完整路径编译代码
