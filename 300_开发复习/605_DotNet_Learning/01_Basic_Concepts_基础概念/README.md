# 基础概念与环境

## 章节介绍

本章节介绍.NET和C#的基础概念，包括.NET平台简介、C#语言简介、开发环境搭建、第一个C#程序、基本数据类型、变量和常量、运算符、输入输出、注释和编码规范等内容。

## 学习目标

- 了解.NET平台和C#语言的基本概念
- 掌握C#的基本数据类型和变量声明
- 熟悉C#的运算符和表达式
- 学习C#的输入输出方法
- 了解C#的编码规范

## 运行方法

### 使用命令行

```bash
# 编译
dotnet build

# 运行
dotnet run
```

### 使用IDE

1. **Visual Studio**：
   - 打开Visual Studio
   - 创建新项目
   - 选择"控制台应用"
   - 复制代码到Program.cs
   - 点击"运行"按钮

2. **Visual Studio Code**：
   - 打开Visual Studio Code
   - 安装C#扩展
   - 打开项目文件夹
   - 按F5运行
