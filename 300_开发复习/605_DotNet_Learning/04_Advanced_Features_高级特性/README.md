# 高级特性

## 章节介绍

本章节介绍C#的高级特性，包括泛型、委托与事件、Lambda表达式、扩展方法、异步编程、反射、特性、动态类型和不安全代码等内容。

## 学习目标

- 掌握C#的泛型编程
- 学习C#的委托与事件
- 熟悉C#的Lambda表达式
- 了解C#的扩展方法
- 掌握C#的异步编程
- 了解C#的反射和特性

## 代码示例

### Program.cs

本示例展示了C#的高级特性，包括：

1. **泛型**：泛型类和泛型方法
2. **委托与事件**：委托定义、事件声明和订阅
3. **Lambda表达式**：简洁的匿名函数
4. **扩展方法**：为现有类型添加新方法
5. **异步编程**：async/await模式
6. **反射**：运行时类型信息和操作
7. **特性**：自定义特性和使用

## 运行方法

### 使用命令行

```bash
# 编译
dotnet build

# 运行
dotnet run
```

### 使用IDE

1. **Visual Studio**：
   - 打开Visual Studio
   - 创建新项目
   - 选择"控制台应用"
   - 复制代码到Program.cs
   - 点击"运行"按钮

2. **Visual Studio Code**：
   - 打开Visual Studio Code
   - 安装C#扩展
   - 打开项目文件夹
   - 按F5运行
