# 02_Installation_安装配置

## 学习目标
- 掌握Elasticsearch的安装
- 理解配置文件
- 学会启动和停止
- 掌握健康检查
- 理解基本配置

## 关键要点
### 1. 环境要求
- Java 8或11
- 内存要求
- 磁盘空间
- 操作系统支持

### 2. 下载安装
- 官网下载
- 解压安装
- 目录结构
- 环境变量

### 3. 配置文件
- elasticsearch.yml
- jvm.options
- log4j2.properties
- 集群名称
- 节点名称
- 网络配置
- 路径配置

### 4. 启动和停止
- 前台启动
- 后台启动
- 停止服务
- 日志查看

### 5. 健康检查
- 访问http://localhost:9200
- 集群健康API
- 节点信息API
- 索引列表API

### 6. 基本配置
- 集群名称
- 节点名称
- 网络主机
- HTTP端口
- 数据和日志路径
- 内存设置
- JVM参数

### 7. 安全配置
- 密码保护
- TLS/SSL
- 角色和权限
- 审计日志

## 实践任务
1. 下载并安装Elasticsearch
2. 配置基本参数
3. 启动Elasticsearch
4. 验证安装成功
