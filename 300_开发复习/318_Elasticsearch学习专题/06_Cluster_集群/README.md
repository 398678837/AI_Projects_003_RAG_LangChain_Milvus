# 06_Cluster_集群

## 学习目标
- 理解集群架构
- 掌握节点类型
- 理解分片和副本
- 学会集群管理
- 理解集群发现

## 关键要点
### 1. 集群架构
- 多个节点组成
- 统一集群名称
- 自动发现
- 主从模式

### 2. 节点类型
- Master节点：管理集群
- Data节点：存储数据
- Client节点：路由请求
- Ingest节点：数据预处理
- Coordinating节点：协调节点
- Machine Learning节点：机器学习

### 3. 分片
- 水平拆分索引
- 主分片（Primary）
- 副分片（Replica）
- 分片数创建时指定
- 提高并行处理

### 4. 副本
- 分片的拷贝
- 提高可用性
- 提高读性能
- 副本数可以动态调整

### 5. 集群发现
- Zen Discovery
- Seed Hosts
- Master选举
- 集群状态同步
- 脑裂问题

### 6. 集群管理
- 集群健康检查
- 节点管理
- 分片分配
- 集群设置
- 滚动升级

### 7. 高可用
- 主分片和副本不在同一节点
- 节点故障自动恢复
- 副本自动提升
- 数据自动重平衡

## 实践任务
1. 配置多节点集群
2. 理解分片分配
3. 监控集群健康
4. 测试故障恢复
