## 关键要点
### 1. Kafka概述
- Apache Kafka
- 分布式流平台
- 高吞吐量
- 低延迟
- 持久化
- 可扩展

### 2. 核心架构
- Producer：生产者
- Consumer：消费者
- Broker：代理
- Topic：主题
- Partition：分区
- Replica：副本
- ZooKeeper/KRaft

### 3. 消息模型
- 发布订阅
- 分区模型
- 消费组
- Offset

### 4. 核心概念
- Topic：主题
- Partition：分区
- Offset：偏移量
- Replica：副本
- Leader：主副本
- Follower：从副本

### 5. 特性
- 高吞吐
- 低延迟
- 持久化
- 可扩展
- 容错
- 多副本

### 6. 应用场景
- 消息系统
- 日志收集
- 用户活动跟踪
- 流处理
- 日志聚合
- 事件源
- 数据管道
