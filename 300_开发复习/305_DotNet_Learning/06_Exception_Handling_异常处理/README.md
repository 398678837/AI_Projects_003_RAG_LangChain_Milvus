# 异常处理

## 章节介绍

本章节介绍C#的异常处理机制，包括异常处理基础、异常的抛出和捕获、标准异常、自定义异常、异常安全、异常处理的最佳实践、异常与构造函数和析构函数、异常与方法签名、异常处理的高级特性和异常处理的实际应用等内容。

## 学习目标

- 掌握C#的异常处理基本语法
- 学习C#的异常抛出和捕获
- 熟悉C#的标准异常类型
- 了解如何创建和使用自定义异常
- 掌握异常处理的最佳实践

## 代码示例

### Program.cs

本示例展示了C#的异常处理机制，包括：

1. **基本异常处理**：try-catch-finally块
2. **多个异常处理**：捕获不同类型的异常
3. **抛出异常**：使用throw语句抛出异常
4. **自定义异常**：创建和使用自定义异常类
5. **异常处理最佳实践**：使用using语句和适当的异常类型

## 运行方法

### 使用命令行

```bash
# 编译
dotnet build

# 运行
dotnet run
```

### 使用IDE

1. **Visual Studio**：
   - 打开Visual Studio
   - 创建新项目
   - 选择"控制台应用"
   - 复制代码到Program.cs
   - 点击"运行"按钮

2. **Visual Studio Code**：
   - 打开Visual Studio Code
   - 安装C#扩展
   - 打开项目文件夹
   - 按F5运行
