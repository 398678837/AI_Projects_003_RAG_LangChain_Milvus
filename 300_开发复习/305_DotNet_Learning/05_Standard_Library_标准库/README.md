# 标准库

## 章节介绍

本章节介绍.NET标准库的使用，包括集合类、字符串处理、日期时间、文件操作、网络编程、正则表达式、序列化、加密解密和并行计算等内容。

## 学习目标

- 掌握.NET集合类的使用
- 学习.NET字符串处理方法
- 熟悉.NET日期时间操作
- 了解.NET文件操作
- 掌握.NET正则表达式
- 了解.NET LINQ的使用

## 代码示例

### Program.cs

本示例展示了.NET标准库的使用，包括：

1. **集合类**：List、Dictionary、HashSet
2. **字符串处理**：长度、大小写转换、替换、分割
3. **日期时间**：当前时间、日期格式化
4. **文件操作**：写入、读取、删除文件
5. **正则表达式**：匹配邮箱和电话号码
6. **LINQ**：过滤、排序、聚合
7. **网络请求**：使用HttpClient发送HTTP请求

## 运行方法

### 使用命令行

```bash
# 编译
dotnet build

# 运行
dotnet run
```

### 使用IDE

1. **Visual Studio**：
   - 打开Visual Studio
   - 创建新项目
   - 选择"控制台应用"
   - 复制代码到Program.cs
   - 点击"运行"按钮

2. **Visual Studio Code**：
   - 打开Visual Studio Code
   - 安装C#扩展
   - 打开项目文件夹
   - 按F5运行
