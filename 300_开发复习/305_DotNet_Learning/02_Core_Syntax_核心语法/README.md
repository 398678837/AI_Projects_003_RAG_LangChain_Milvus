# 核心语法

## 章节介绍

本章节介绍C#的核心语法，包括控制流、函数、数组、字符串、指针与引用、命名空间、类型定义、类型推断、初始化和LINQ基础等内容。

## 学习目标

- 掌握C#的控制流语句（if-else、switch、循环）
- 学习C#的函数定义和调用
- 熟悉C#的数组操作
- 了解C#的字符串处理方法
- 掌握C#的命名空间和类型定义
- 了解LINQ基础

## 运行方法

### 使用命令行

```bash
# 编译
dotnet build

# 运行
dotnet run
```

### 使用IDE

1. **Visual Studio**：
   - 打开Visual Studio
   - 创建新项目
   - 选择"控制台应用"
   - 复制代码到Program.cs
   - 点击"运行"按钮

2. **Visual Studio Code**：
   - 打开Visual Studio Code
   - 安装C#扩展
   - 打开项目文件夹
   - 按F5运行
