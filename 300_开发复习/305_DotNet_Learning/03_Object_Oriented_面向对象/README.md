# 面向对象编程

## 章节介绍

本章节介绍C#的面向对象编程特性，包括类和对象、构造函数和析构函数、继承、多态、封装、抽象、静态成员、接口、运算符重载和特殊成员函数等内容。

## 学习目标

- 掌握C#的类和对象概念
- 学习C#的继承和多态特性
- 熟悉C#的封装和抽象
- 了解C#的接口和静态成员
- 掌握C#的运算符重载

## 代码示例

### Program.cs

本示例展示了C#的面向对象编程特性，包括：

1. **类和对象**：类定义和对象创建
2. **继承**：基类和派生类
3. **多态**：虚方法和方法重写
4. **抽象类和接口**：抽象方法和接口实现
5. **封装**：私有字段和公共属性
6. **静态成员**：静态属性和方法

## 运行方法

### 使用命令行

```bash
# 编译
dotnet build

# 运行
dotnet run
```

### 使用IDE

1. **Visual Studio**：
   - 打开Visual Studio
   - 创建新项目
   - 选择"控制台应用"
   - 复制代码到Program.cs
   - 点击"运行"按钮

2. **Visual Studio Code**：
   - 打开Visual Studio Code
   - 安装C#扩展
   - 打开项目文件夹
   - 按F5运行
