# 01_Basic_Concepts_基础概念

## 学习目标
- 理解RabbitMQ的基本概念
- 掌握AMQP协议
- 理解核心架构
- 理解消息模型
- 掌握应用场景

## 关键要点
### 1. RabbitMQ概述
- 开源消息中间件
- 基于AMQP协议
- Erlang开发
- 高可靠性
- 灵活的路由

### 2. 核心架构
- Producer：生产者
- Consumer：消费者
- Broker：消息代理
- Exchange：交换机
- Queue：队列
- Binding：绑定
- Virtual Host：虚拟主机

### 3. AMQP协议
- Advanced Message Queuing Protocol
- 二进制协议
- 消息模型
- 命令集
- 安全机制

### 4. 消息模型
- 简单队列
- 工作队列
- 发布订阅
- 路由
- 主题
- RPC

### 5. 核心概念
- Exchange类型：Direct、Topic、Fanout、Headers
- Queue：消息队列
- Binding：绑定关系
- Routing Key：路由键
- Message：消息

### 6. 应用场景
- 异步处理
- 流量削峰
- 日志收集
- 系统解耦
- 分布式事务
