# 05_Collections_集合

- 掌握序列(Sequence)

1. **集合类型**：

   - Map
2. **不可变/可变**：
   - 不可变集合
   - 可变集合
3. **集合操作**：
   - 转换
   - 过滤
   - 映射
   - 归约
4. **序列**：
   - Sequence创建
   - 序列操作
   - 惰性求值
