# 01_Basic_Concepts_基础概念

## 学习目标
- 理解Kotlin的基本概念
- 掌握变量和常量的定义
- 学会基本数据类型
- 理解Kotlin与Java的关系

## 知识点总结
1. **Kotlin简介**：
   - JetBrains开发
   - 100%兼容Java
   - 简洁、安全、互操作
2. **变量和常量**：
   - var：可变变量
   - val：只读变量
   - 类型推断
3. **基本数据类型**：
   - 数字类型
   - 布尔类型
   - 字符类型
   - 字符串类型
4. **第一个程序**：
   - main函数
   - Hello World
