# 08_Android_Android开发

## 学习目标
- 掌握Kotlin Android开发基础
- 理解Activity和Fragment
- 学会Jetpack组件
- 掌握Coroutines在Android中的应用

## 知识点总结
1. **Android基础**：
   - Activity
   - Fragment
   - 布局
   - 资源
2. **Jetpack组件**：
   - ViewModel
   - LiveData
   - Room
   - DataStore
3. **网络请求**：
   - Retrofit
   - OkHttp
   - Kotlin Coroutines
4. **UI框架**：
   - ViewBinding
   - Jetpack Compose
