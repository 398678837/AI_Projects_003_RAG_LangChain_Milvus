# 05_Package_Management_包管理

## 学习目标
- 理解包的概念和结构
- 掌握package.json的配置
- 学会如何发布包
- 了解包的版本管理

## 知识点总结
1. **包的结构**：
   - package.json
   - 入口文件
   - 源代码目录
   - 测试目录
2. **package.json配置**：
   - 基本信息：name, version, description
   - 脚本：scripts
   - 依赖：dependencies, devDependencies
   - 入口：main, module, types
3. **包的发布**：
   - npm用户注册
   - npm login
   - npm version
   - npm publish
4. **包的版本管理**：
   - 语义化版本
   - 预发布版本
   - 标签（tag）
