# 03_Yarn_Yarn

## 学习目标
- 掌握Yarn的常用命令
- 理解Yarn的特点和优势
- 学会使用Yarn管理依赖
- 了解Yarn的高级功能

## 知识点总结
1. **Yarn简介**：
   - Facebook开发的包管理器
   - 注重速度、安全和可靠性
   - 兼容npm注册表
2. **Yarn特点**：
   - 离线模式
   - 确定性安装
   - 网络性能优化
   - 并行下载
3. **常用命令**：
   - yarn init：初始化
   - yarn add：添加依赖
   - yarn install：安装依赖
   - yarn workspaces：Monorepo
4. **Yarn 2/3（Berry）**：
   - 全新架构
   - Plug'n'Play
   - Zero-Installs
