# 02_NPM_NPM

## 学习目标
- 掌握npm的常用命令
- 理解npm的工作原理
- 学会使用npm管理依赖
- 了解npm的配置和高级用法

## 知识点总结
1. **npm简介**：
   - Node.js官方包管理器
   - 最大的开源软件注册表
   - 超过200万个包
2. **常用命令**：
   - npm init：初始化项目
   - npm install：安装依赖
   - npm run：运行脚本
   - npm publish：发布包
3. **npm配置**：
   - .npmrc配置文件
   - 镜像源配置
   - 代理配置
4. **高级功能**：
   - npm workspaces
   - npm scripts
   - npx
   - npm link
