# 07_Scripts_脚本

## 学习目标
- 掌握npm scripts的使用
- 理解脚本的生命周期
- 学会编写复杂的脚本
- 了解脚本的最佳实践

## 知识点总结
1. **npm scripts简介**：
   - 在package.json中定义
   - 通过npm run运行
   - 可以执行任何命令
2. **常用脚本**：
   - start：启动应用
   - dev：开发模式
   - build：构建
   - test：测试
   - lint：代码检查
3. **脚本生命周期**：
   - pre/post前缀
   - 自动执行
4. **高级用法**：
   - 变量传递
   - 并行执行
   - 串行执行
   - 跨平台脚本
