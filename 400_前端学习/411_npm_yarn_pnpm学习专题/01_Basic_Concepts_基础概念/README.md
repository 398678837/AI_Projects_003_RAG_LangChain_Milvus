# 01_Basic_Concepts_基础概念

## 学习目标
- 理解包管理器的作用
- 掌握npm/yarn/pnpm的基本概念
- 了解包管理器的发展历史
- 理解package.json的结构

## 知识点总结
1. **包管理器简介**：
   - 管理项目依赖
   - 自动下载和安装包
   - 处理版本冲突
   - 提供脚本运行功能
2. **常见包管理器**：
   - npm：Node.js官方包管理器
   - Yarn：Facebook开发的包管理器
   - pnpm：快速、节省磁盘空间的包管理器
3. **核心概念**：
   - 包（Package）
   - 依赖（Dependency）
   - 版本（Version）
   - 注册表（Registry）
   - lock文件
4. **package.json**：
   - 项目元数据
   - 依赖列表
   - 脚本命令
   - 配置信息
