# 04_Pnpm_Pnpm

## 学习目标
- 掌握pnpm的常用命令
- 理解pnpm的硬链接机制
- 学会使用pnpm管理依赖
- 了解pnpm的优势

## 知识点总结
1. **pnpm简介**：
   - 快速、节省磁盘空间的包管理器
   - 使用硬链接和符号链接
   - 严格的依赖管理
2. **pnpm核心特点**：
   - 内容寻址存储
   - 硬链接节省空间
   - 严格的node_modules结构
   - 非扁平依赖树
3. **常用命令**：
   - pnpm init：初始化
   - pnpm add：添加依赖
   - pnpm install：安装依赖
   - pnpm workspace：Monorepo
4. **性能优势**：
   - 磁盘空间节省90%
   - 安装速度极快
   - 严格的依赖解析
