## 知识点总结
1. **认证方式总结**：
   - Cookie + Session：传统Web应用
   - Token/JWT：SPA、移动App、微服务
   - OAuth2：第三方登录
2. **安全原则**：
   - HTTPS、HttpOnly、SameSite、CSP
   - 密码加盐哈希
   - 最小权限、纵深防御
3. **实战应用**：
   - 前后端分离
   - 微服务架构
   - 第三方登录
4. **学习路径**：
   - 基础 → Cookie → Token → JWT → Session → OAuth2 → 安全 → 最佳实践 → 实战
