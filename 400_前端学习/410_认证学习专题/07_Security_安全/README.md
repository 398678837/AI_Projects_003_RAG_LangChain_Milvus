# 07_Security_安全

## 学习目标
- 理解认证相关的安全威胁
- 掌握常见的安全防护措施
- 学会实现安全的认证系统
- 理解密码安全和哈希算法

## 知识点总结
1. **常见安全威胁**：
   - XSS：跨站脚本攻击
   - CSRF：跨站请求伪造
   - 密码破解
   - 会话劫持
   - MITM：中间人攻击
2. **密码安全**：
   - 加盐哈希
   - bcrypt/argon2/scrypt
   - 密码强度要求
   - 密码存储最佳实践
3. **HTTPS**：
   - TLS/SSL加密
   - 证书验证
   - HSTS
4. **其他安全措施**：
   - 速率限制
   - 验证码
   - 多因素认证
   - 安全日志
   - 审计
