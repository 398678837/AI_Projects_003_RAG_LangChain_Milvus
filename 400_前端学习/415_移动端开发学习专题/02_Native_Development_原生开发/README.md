# 02_Native_Development_原生开发

## 学习目标
- 理解原生开发的概念
- 掌握iOS和Android原生开发基础
- 了解Swift和Kotlin基础语法
- 理解原生开发的优缺点

## 知识点总结
1. **iOS原生开发**：
   - Swift语法基础
   - UIKit/SwiftUI
   - Xcode使用
2. **Android原生开发**：
   - Kotlin语法基础
   - Jetpack Compose
   - Android Studio使用
3. **原生开发优势**：
   - 最佳性能
   - 最佳体验
   - 完整API访问
4. **开发流程**：
   - 环境配置
   - 项目创建
   - 调试发布
