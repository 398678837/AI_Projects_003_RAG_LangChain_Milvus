# 06_Performance_Optimization_性能优化

## 学习目标
- 掌握移动端性能优化技巧
- 理解启动优化
- 学会渲染优化
- 掌握网络和存储优化

## 知识点总结
1. **启动优化**：
   - 启动时间测量
   - 代码分包
   - 资源预加载
2. **渲染优化**：
   - 列表虚拟化
   - 懒加载
   - 减少重绘
3. **内存优化**：
   - 内存泄漏检测
   - 图片优化
   - 缓存策略
4. **网络优化**：
   - 请求合并
   - 数据缓存
   - 图片优化
