# 09_Security_安全

## 学习目标
- 掌握移动端安全最佳实践
- 理解数据加密和安全存储
- 学会网络通信安全
- 掌握代码安全和反逆向

## 知识点总结
1. **数据安全**：
   - 敏感数据加密
   - 安全存储
   - 数据清除
2. **网络安全**：
   - HTTPS/TLS
   - 证书校验
   - 请求加密
3. **代码安全**：
   - 代码混淆
   - 反调试
   - 完整性校验
4. **认证安全**：
   - 安全认证
   - Token管理
   - 会话管理
