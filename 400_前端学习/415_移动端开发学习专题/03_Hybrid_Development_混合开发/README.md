# 03_Hybrid_Development_混合开发

## 学习目标
- 理解混合开发的概念
- 掌握Cordova/Capacitor的使用
- 理解Ionic框架
- 掌握混合开发的优缺点

## 知识点总结
1. **混合开发原理**：
   - WebView容器
   - 原生桥接
   - 插件系统
2. **Cordova/Capacitor**：
   - 项目创建
   - 插件使用
   - 原生能力调用
3. **Ionic框架**：
   - UI组件
   - 项目结构
   - 路由导航
4. **优缺点**：
   - 开发效率
   - 性能对比
   - 适用场景
