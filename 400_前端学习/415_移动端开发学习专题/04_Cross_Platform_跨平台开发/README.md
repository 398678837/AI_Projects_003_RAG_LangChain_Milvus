# 04_Cross_Platform_跨平台开发

## 学习目标
- 理解跨平台开发的概念
- 掌握React Native和Flutter
- 理解uni-app的使用
- 掌握跨平台开发的优缺点

## 知识点总结
1. **React Native**：
   - React基础
   - 组件和API
   - 原生模块
2. **Flutter**：
   - Dart基础
   - Widget系统
   - 状态管理
3. **uni-app**：
   - Vue基础
   - 多端发布
   - 组件库
4. **技术对比**：
   - 性能对比
   - 生态对比
   - 学习曲线
