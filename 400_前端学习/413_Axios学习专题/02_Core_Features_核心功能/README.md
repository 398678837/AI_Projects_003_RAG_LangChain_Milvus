# 02_Core_Features_核心功能

## 学习目标
- 掌握Axios的核心功能
- 理解请求和响应转换
- 学会取消请求
- 掌握文件上传

## 知识点总结
1. **请求方法别名**：
   - axios.request()
   - axios.get()
   - axios.post()
   - axios.put()
   - axios.patch()
   - axios.delete()
   - axios.head()
   - axios.options()
2. **请求配置**：
   - url
   - method
   - baseURL
   - headers
   - params
   - data
3. **并发请求**：
   - axios.all()
   - axios.spread()
   - Promise.all()
