# 10_Real_World_实战应用

## 学习目标
- 掌握Axios在实际项目中的应用
- 理解完整的请求流程
- 学会常见场景的处理
- 掌握项目级别的封装

## 知识点总结
1. **用户认证**：
   - 登录请求
   - Token存储
   - Token刷新
2. **数据管理**：
   - 列表查询
   - 详情查询
   - 增删改查
3. **文件上传**：
   - 单文件上传
   - 多文件上传
   - 上传进度
4. **下载文件**：
   - 单文件下载
   - 下载进度
5. **WebSocket**：
   - 连接管理
   - 消息收发
