# 03_Request_Configuration_请求配置

## 学习目标
- 掌握Axios的请求配置
- 理解全局默认配置
- 学会配置baseURL
- 掌握请求拦截器

## 知识点总结
1. **请求配置对象**：
   - url
   - method
   - baseURL
   - headers
   - params
   - data
   - timeout
   - auth
2. **全局默认配置**：
   - axios.defaults
   - 全局baseURL
   - 全局headers
3. **请求配置优先级**：
   - 实例配置 > 全局配置 > 默认配置
