# 05_Error_Handling_错误处理

## 学习目标
- 掌握Axios的错误处理
- 理解错误对象结构
- 学会不同类型错误的处理
- 掌握重试机制

## 知识点总结
1. **错误类型**：
   - 请求错误
   - 响应错误
   - 网络错误
   - 超时错误
2. **错误对象**：
   - error.response
   - error.request
   - error.message
   - error.config
3. **处理策略**：
   - 统一错误处理
   - 错误提示
   - 错误日志
   - 重试机制
