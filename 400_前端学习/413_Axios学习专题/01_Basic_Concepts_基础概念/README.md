# 01_Basic_Concepts_基础概念

## 学习目标
- 理解Axios是什么
- 掌握Axios的安装和引入
- 学会基本的HTTP请求
- 理解Promise基础

## 知识点总结
1. **Axios简介**：
   - 基于Promise的HTTP客户端
   - 浏览器和Node.js都可用
   - 支持请求和响应拦截器
2. **安装Axios**：
   - npm install axios
   - yarn add axios
   - pnpm add axios
3. **基本请求**：
   - axios.get()
   - axios.post()
   - axios.put()
   - axios.delete()
4. **响应处理**：
   - .then()/.catch()
   - async/await
   - 响应结构
