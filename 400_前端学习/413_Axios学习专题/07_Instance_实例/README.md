# 07_Instance_实例

## 学习目标
- 掌握创建Axios实例
- 理解实例配置
- 学会多个实例的使用
- 掌握实例的继承

## 知识点总结
1. **创建实例**：
   - axios.create()
   - 配置实例
2. **实例配置**：
   - baseURL
   - timeout
   - headers
3. **多个实例**：
   - API实例
   - 文件上传实例
   - 第三方API实例
4. **实例方法**：
   - get/post/put/delete
   - 与全局方法相同
