# 05_Database_数据库

## 学习目标
- 掌握Node.js中数据库操作的基本方法
- 理解MongoDB、MySQL、SQLite和Redis的使用
- 熟悉数据库的增删改查操作
- 学会使用不同类型的数据库

## 知识点总结
1. **MongoDB**：NoSQL数据库，使用mongoose进行操作
2. **MySQL**：关系型数据库，使用mysql2进行操作
3. **SQLite**：轻量级关系型数据库，使用sqlite3进行操作
4. **Redis**：内存数据库，用于缓存和会话管理
5. **数据库操作**：增删改查（CRUD）
6. **连接管理**：创建和关闭数据库连接
7. **错误处理**：处理数据库操作中的错误
