# 09_Deployment_部署

## 学习目标
- 理解Node.js应用的部署流程
- 掌握环境变量配置和生产环境优化
- 熟悉Docker和容器化部署
- 学会使用PM2进行进程管理

## 知识点总结
1. **环境变量配置**：使用dotenv管理环境变量
2. **生产环境中间件**：日志、安全、压缩等中间件
3. **静态文件服务**：优化静态文件的访问
4. **Docker部署**：使用Dockerfile和docker-compose
5. **PM2进程管理**：使用PM2管理Node.js进程
6. **CI/CD**：持续集成和持续部署
7. **部署脚本**：自动化部署流程
