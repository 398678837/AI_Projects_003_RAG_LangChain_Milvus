# 03_Asynchronous_异步编程

## 学习目标
- 理解Node.js的异步编程模型
- 掌握回调函数、Promise和Async/Await
- 熟悉事件循环和异步操作的执行顺序
- 学会处理异步错误和并发操作

## 知识点总结
1. **回调函数**：传统的异步处理方式
2. **Promise**：更优雅的异步处理方式
3. **Async/Await**：基于Promise的语法糖
4. **事件循环**：Node.js的事件处理机制
5. **并行异步操作**：使用Promise.all
6. **串行异步操作**：使用async/await
7. **错误处理**：try/catch/finally
8. **Promise链式调用**：处理多个异步操作
