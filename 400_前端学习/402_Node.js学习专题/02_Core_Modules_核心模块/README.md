# 02_Core_Modules_核心模块

## 学习目标
- 掌握Node.js核心模块的使用
- 理解文件系统、HTTP服务器和路径处理
- 熟悉事件处理和操作系统信息获取
- 学会使用加密和压缩功能

## 知识点总结
1. **fs 模块**：文件系统操作，包括读取、写入、创建目录等
2. **http 模块**：创建HTTP服务器和客户端
3. **path 模块**：处理文件路径
4. **url 模块**：解析和处理URL
5. **querystring 模块**：处理查询字符串
6. **events 模块**：事件处理
7. **os 模块**：获取操作系统信息
8. **process 模块**：获取进程信息
9. **crypto 模块**：加密功能
10. **zlib 模块**：压缩功能
