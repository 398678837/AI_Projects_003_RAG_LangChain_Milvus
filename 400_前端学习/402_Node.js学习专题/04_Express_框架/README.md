# 04_Express_框架

## 学习目标
- 理解Express框架的基本概念和工作原理
- 掌握Express的路由和中间件
- 熟悉静态文件服务和模板引擎
- 学会处理错误和404情况

## 知识点总结
1. **基本设置**：创建Express应用，设置端口
2. **中间件**：使用内置中间件，创建自定义中间件
3. **路由**：定义GET、POST、PUT、DELETE等路由
4. **参数处理**：路径参数和查询参数
5. **静态文件服务**：提供静态文件访问
6. **错误处理**：错误处理中间件
7. **404处理**：处理未找到的路由
8. **路由模块化**：使用Router创建模块化路由
9. **模板引擎**：使用EJS等模板引擎
10. **中间件链**：多个中间件的顺序执行
