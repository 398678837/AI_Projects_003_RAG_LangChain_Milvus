# 01_Basic_Concepts_基础概念

## 学习目标
- 理解Node.js的基本概念和工作原理
- 掌握Node.js的模块系统
- 熟悉文件系统操作和HTTP服务器创建
- 学会使用事件处理和错误处理

## 知识点总结
1. **模块系统**：CommonJS模块系统，require和module.exports
2. **文件系统**：fs模块，同步和异步操作
3. **HTTP服务器**：http模块，创建简单的Web服务器
4. **事件处理**：EventEmitter类，事件的触发和监听
5. **路径处理**：path模块，处理文件路径
6. **环境变量**：process对象，获取环境信息
7. **错误处理**：try/catch，错误处理机制
8. **包管理**：package.json，npm包管理
