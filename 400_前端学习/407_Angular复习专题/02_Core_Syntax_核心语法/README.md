# 02_Core_Syntax_核心语法

## 学习目标
- 掌握Angular模板语法
- 理解插值表达式、属性绑定、事件绑定
- 学会使用结构指令和属性指令
- 掌握管道的使用方法

## 知识点总结
1. **插值表达式**：使用 {{ }} 在模板中显示组件数据
2. **属性绑定**：使用 [ ] 将数据绑定到DOM元素属性
3. **事件绑定**：使用 ( ) 监听DOM事件
4. **双向绑定**：使用 [(ngModel)] 实现数据双向绑定
5. **结构指令**：*ngIf、*ngFor、*ngSwitch 用于控制DOM结构
6. **属性指令**：ngClass、ngStyle 用于修改元素样式
7. **管道**：用于转换数据显示格式，如日期、货币、大小写等
