# 01_Basic_Concepts_基础概念

## 学习目标
- 理解Angular的基本概念和架构
- 掌握Angular项目的创建和运行
- 熟悉Angular的核心概念：模块、组件、模板、服务
- 了解Angular CLI的基本使用

## 知识点总结
1. **Angular简介**：由Google开发的前端框架，用于构建单页应用（SPA）
2. **Angular CLI**：命令行工具，用于创建项目、生成代码、构建和测试
3. **模块（NgModule）**：Angular应用的基本构建块，用于组织代码
4. **组件（Component）**：视图和逻辑的结合体，是Angular应用的核心
5. **模板（Template）**：定义组件的视图，使用HTML和Angular语法
6. **数据绑定**：组件和模板之间的数据交互方式
