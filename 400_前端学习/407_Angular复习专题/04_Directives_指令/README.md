# 04_Directives_指令

## 学习目标
- 理解指令的概念和分类
- 掌握内置指令的使用
- 学会创建自定义属性指令
- 学会创建自定义结构指令

## 知识点总结
1. **指令分类**：
   - 组件指令：带有模板的指令
   - 属性指令：改变元素外观或行为的指令
   - 结构指令：改变DOM结构的指令
2. **内置属性指令**：ngClass、ngStyle、ngModel
3. **内置结构指令**：*ngIf、*ngFor、*ngSwitch
4. **自定义属性指令**：使用 @Directive 装饰器，通过 ElementRef 和 Renderer2 操作DOM
5. **自定义结构指令**：使用 TemplateRef 和 ViewContainerRef 操作视图
