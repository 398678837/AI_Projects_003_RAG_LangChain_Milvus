# 07_SwiftUI_SwiftUI

## 学习目标
- 理解SwiftUI的声明式语法
- 掌握常用视图和控件
- 理解状态管理（@State、@Binding、@ObservedObject、@EnvironmentObject）
- 掌握导航和列表
- 理解SwiftUI与UIKit的互操作

## 关键要点
### 1. SwiftUI概述
- 声明式UI框架
- 跨平台（iOS、macOS、watchOS、tvOS）
- 自动响应数据变化
- 预览功能

### 2. 基础视图
- Text：文本
- Image：图片
- Button：按钮
- TextField：输入框
- Toggle：开关
- Slider：滑块
- Picker：选择器

### 3. 布局容器
- VStack：垂直栈
- HStack：水平栈
- ZStack：层叠栈
- List：列表
- Form：表单
- ScrollView：滚动视图
- LazyVStack/LazyHStack：懒加载栈

### 4. 状态管理
- @State：视图内部状态
- @Binding：父子视图双向绑定
- @ObservedObject：观察对象
- @StateObject：拥有对象
- @EnvironmentObject：环境对象
- @Environment：环境变量

### 5. 导航
- NavigationStack/NavigationView
- navigationDestination
- sheet
- fullScreenCover

### 6. 列表和数据
- List
- ForEach
- Identifiable协议

## 实践任务
1. 创建SwiftUI应用
2. 实现列表和导航
3. 使用状态管理
4. 与UIKit集成
