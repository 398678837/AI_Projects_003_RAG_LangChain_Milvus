# 06_AppKit_AppKit

## 学习目标
- 理解AppKit框架
- 掌握NSWindow和NSViewController
- 学会使用常用控件
- 理解响应者链
- 掌握Mac应用开发流程

## 关键要点
### 1. AppKit概述
- macOS UI框架
- 基于UIKit，但有很多不同
- 支持多窗口
- 菜单栏、工具栏

### 2. NSApplication
- Mac应用中心
- NSApplicationDelegate
- 应用生命周期

### 3. NSWindow
- 窗口
- NSWindowController
- 窗口样式

### 4. NSView
- 视图基类
- drawRect:
- Autolayout

### 5. 常用控件
- NSButton
- NSTextField
- NSTableView
- NSCollectionView

## 实践任务
1. 创建Mac应用
2. 使用NSWindow
3. 实现简单的Mac界面
