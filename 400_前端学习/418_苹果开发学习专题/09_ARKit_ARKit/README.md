# 09_ARKit_ARKit

## 学习目标
- 理解ARKit的基本概念
- 掌握ARSCNView的使用
- 学会平面检测
- 理解3D物体渲染
- 掌握ARSession配置

## 关键要点
### 1. ARKit概述
- 苹果增强现实框架
- 结合设备摄像头和运动传感器
- 支持3D物体放置
- 真实感渲染

### 2. 核心组件
- ARSession：AR会话
- ARSCNView：AR视图（SceneKit）
- ARSKView：AR视图（SpriteKit）
- ARConfiguration：配置

### 3. 配置类型
- ARWorldTrackingConfiguration：世界追踪
- AROrientationTrackingConfiguration：方向追踪
- ARFaceTrackingConfiguration：人脸追踪
- ARBodyTrackingConfiguration：身体追踪

### 4. 平面检测
- 水平平面
- 垂直平面
- ARPlaneAnchor

### 5. 3D内容
- SceneKit：SCNNode
- RealityKit：Entity
- 添加物体
- 交互操作

## 实践任务
1. 创建AR应用
2. 检测平面
3. 放置3D物体
4. 实现交互
