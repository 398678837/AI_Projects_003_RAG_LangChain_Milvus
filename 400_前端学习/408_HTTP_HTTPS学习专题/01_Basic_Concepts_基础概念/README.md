# 01_Basic_Concepts_基础概念

## 学习目标
- 理解HTTP和HTTPS的基本概念
- 了解网络模型和协议层次
- 掌握URL的组成结构
- 理解客户端-服务器架构

## 知识点总结
1. **HTTP简介**：超文本传输协议，用于在客户端和服务器之间传输数据
2. **HTTPS简介**：HTTP的安全版本，通过SSL/TLS加密数据传输
3. **网络模型**：OSI七层模型、TCP/IP四层模型
4. **URL结构**：协议、域名、端口、路径、查询参数、片段
5. **客户端-服务器架构**：浏览器作为客户端，Web服务器提供服务
6. **请求-响应模式**：客户端发送请求，服务器返回响应
