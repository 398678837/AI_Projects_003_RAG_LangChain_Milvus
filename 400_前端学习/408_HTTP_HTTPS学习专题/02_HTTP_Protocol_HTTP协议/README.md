# 02_HTTP_Protocol_HTTP协议

## 学习目标
- 理解HTTP协议的工作原理
- 掌握HTTP请求和响应的结构
- 了解HTTP的版本演进
- 理解无状态和幂等性概念

## 知识点总结
1. **HTTP请求结构**：
   - 请求行：方法、路径、协议版本
   - 请求头：键值对形式的元数据
   - 空行：分隔头和体
   - 请求体：可选的数据
2. **HTTP响应结构**：
   - 状态行：协议版本、状态码、状态消息
   - 响应头：键值对形式的元数据
   - 空行：分隔头和体
   - 响应体：返回的数据
3. **HTTP版本**：
   - HTTP/1.0：每次请求建立新连接
   - HTTP/1.1：持久连接、流水线
   - HTTP/2：二进制格式、多路复用
   - HTTP/3：基于QUIC协议
4. **无状态协议**：每个请求独立，服务器不保留状态
5. **幂等性**：多次执行结果相同
