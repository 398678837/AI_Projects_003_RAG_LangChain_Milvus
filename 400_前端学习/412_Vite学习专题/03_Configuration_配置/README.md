# 03_Configuration_配置

## 学习目标
- 掌握vite.config.js配置
- 理解各种配置选项
- 学会路径别名配置
- 掌握构建配置

## 知识点总结
1. **配置文件**：
   - vite.config.js/ts
   - defineConfig辅助函数
   - 类型提示
2. **基础配置**：
   - root：项目根目录
   - base：部署基础路径
   - publicDir：静态资源目录
3. **服务器配置**：
   - server.port：端口
   - server.proxy：代理
   - server.open：自动打开
4. **构建配置**：
   - build.outDir：输出目录
   - build.assetsDir：资源目录
   - build.sourcemap：源码映射
5. **路径别名**：
   - resolve.alias
   - 与tsconfig.json配合
