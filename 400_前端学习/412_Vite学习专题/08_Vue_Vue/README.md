# 08_Vue_Vue

## 学习目标
- 掌握Vite中Vue的配置
- 理解Vue插件的使用
- 学会单文件组件
- 掌握Vue项目最佳实践

## 知识点总结
1. **创建Vue项目**：
   - npm create vite@latest -- --template vue
   - npm create vite@latest -- --template vue-ts
2. **Vue插件**：
   - @vitejs/plugin-vue
   - 模板编译配置
3. **单文件组件**：
   - script setup
   - style scoped
4. **环境变量**：
   - VITE_前缀
   - import.meta.env
