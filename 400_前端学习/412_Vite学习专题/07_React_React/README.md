# 07_React_React

## 学习目标
- 掌握Vite中React的配置
- 理解React插件的使用
- 学会Fast Refresh
- 掌握React项目最佳实践

## 知识点总结
1. **创建React项目**：
   - npm create vite@latest -- --template react
   - npm create vite@latest -- --template react-ts
2. **React插件**：
   - @vitejs/plugin-react
   - @vitejs/plugin-react-swc
3. **Fast Refresh**：
   - 保持组件状态
   - 即时更新
4. **环境变量**：
   - VITE_前缀
   - import.meta.env
