# 01_Basic_Concepts_基础概念

## 学习目标
- 理解Vite的设计理念
- 掌握Vite的核心特点
- 学会创建Vite项目
- 理解Vite与Webpack的区别

## 知识点总结
1. **Vite简介**：
   - 下一代前端开发与构建工具
   - 基于ES modules
   - 开发服务器极速启动
2. **Vite核心特点**：
   - 极速的冷启动
   - 即时的热更新（HMR）
   - 原生ES模块支持
   - 丰富的插件生态
3. **创建项目**：
   - npm create vite@latest
   - 选择框架（React/Vue/Svelte等）
4. **开发与构建**：
   - npm run dev：开发模式
   - npm run build：生产构建
   - npm run preview：预览构建结果
5. **Vite vs Webpack**：
   - 开发体验更好
   - 配置更简单
   - 构建速度更快
