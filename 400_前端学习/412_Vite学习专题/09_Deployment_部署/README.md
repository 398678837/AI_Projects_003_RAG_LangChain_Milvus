# 09_Deployment_部署

## 学习目标
- 掌握Vite项目的构建
- 理解不同平台的部署方法
- 学会配置部署环境
- 掌握CI/CD流程

## 知识点总结
1. **构建项目**：
   - npm run build
   - 检查构建产物
   - 本地预览
2. **部署到静态托管**：
   - GitHub Pages
   - Vercel
   - Netlify
   - 阿里云/腾讯云
3. **Nginx部署**：
   - 配置文件
   - 反向代理
   - Gzip压缩
4. **CI/CD**：
   - GitHub Actions
   - GitLab CI
   - 自动化部署
