# 02_Core_Features_核心功能

## 学习目标
- 掌握Vite的核心功能
- 理解HMR的工作原理
- 学会使用静态资源处理
- 掌握环境变量配置

## 知识点总结
1. **开发服务器**：
   - 极速启动
   - 自动打开浏览器
   - 端口自动占用
2. **热模块替换（HMR）**：
   - 即时更新
   - 保持应用状态
   - 框架特定优化
3. **静态资源处理**：
   - 导入资源为URL
   - public目录
   - 资源内联
4. **环境变量**：
   - .env文件
   - import.meta.env
   - 模式（development/production）
5. **TypeScript支持**：
   - 开箱即用
   - 类型检查
   - 路径别名
