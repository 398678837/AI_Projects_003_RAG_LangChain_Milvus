# 06_Typescript_TypeScript

## 学习目标
- 掌握Vite中TypeScript的配置
- 理解类型声明
- 学会路径别名与TypeScript的配合
- 掌握类型安全实践

## 知识点总结
1. **TypeScript支持**：
   - 开箱即用
   - 无需额外配置
   - tsconfig.json
2. **类型声明**：
   - vite-env.d.ts
   - 自定义类型
   - 第三方库类型
3. **路径别名**：
   - vite.config.js + tsconfig.json
   - 类型安全的导入
4. **类型检查**：
   - 构建时类型检查
   - 开发时类型提示
