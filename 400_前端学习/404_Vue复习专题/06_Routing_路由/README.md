# 06_Routing_路由

## 学习目标
- 理解Vue Router的基本概念
- 掌握路由配置和导航
- 熟悉路由参数和守卫
- 学会使用编程式导航

## 知识点总结
1. **路由配置**：定义路由规则和组件映射
2. **路由导航**：使用router-link和编程式导航
3. **路由视图**：使用router-view显示组件
4. **路由参数**：动态路由和查询参数
5. **路由守卫**：全局守卫、路由独享守卫和组件内守卫
6. **路由模式**：hash模式和history模式
7. **嵌套路由**：多级路由结构
8. **命名路由**：使用名称导航
