# 03_Components_组件

## 学习目标
- 理解Vue组件的概念和使用方法
- 掌握组件的传参和事件机制
- 熟悉组件的插槽和作用域插槽
- 学会使用动态组件和异步组件

## 知识点总结
1. **基本组件**：创建和使用基本组件
2. **组件传参**：通过props向组件传递数据
3. **组件事件**：通过事件向父组件传递数据
4. **组件插槽**：使用插槽自定义组件内容
5. **作用域插槽**：在插槽中使用组件的数据
6. **动态组件**：根据条件动态切换组件
7. **异步组件**：异步加载组件
8. **组件通信**：父组件和子组件之间的通信
9. **组件生命周期**：组件的生命周期钩子
10. **组件样式**：组件的样式管理
