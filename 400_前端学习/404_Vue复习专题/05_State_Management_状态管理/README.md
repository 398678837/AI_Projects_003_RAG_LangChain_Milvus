# 05_State_Management_状态管理

## 学习目标
- 理解Vue的状态管理方式
- 掌握组件间状态传递的方法
- 熟悉Vuex的使用
- 学会使用Composition API进行状态管理

## 知识点总结
1. **组件内部状态**：使用data属性管理组件内部状态
2. **父组件向子组件传递状态**：使用props
3. **子组件向父组件传递状态**：使用事件
4. **兄弟组件间状态传递**：通过父组件作为中间层
5. **Vuex**：全局状态管理
6. **Composition API**：使用ref和reactive进行状态管理
7. **Provide/Inject**：跨组件层级传递状态
