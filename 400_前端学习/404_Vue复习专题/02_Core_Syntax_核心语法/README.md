# 02_Core_Syntax_核心语法

## 学习目标
- 掌握Vue的核心语法
- 理解Vue的模板语法和指令
- 熟悉计算属性和监听属性的使用
- 学会使用Vue的修饰符和过滤器

## 知识点总结
1. **模板语法**：文本插值、原始HTML、属性绑定
2. **指令**：v-if、v-for、v-on、v-model等
3. **计算属性**：基于响应式数据计算派生值
4. **监听属性**：监听数据变化并执行操作
5. **方法**：定义和调用组件方法
6. **修饰符**：事件修饰符、按键修饰符、系统修饰符
7. **过滤器**：格式化数据显示
8. **计算属性 vs 方法**：理解两者的区别
9. **响应式原理**：Vue的响应式系统
10. **生命周期**：组件生命周期的各个阶段
