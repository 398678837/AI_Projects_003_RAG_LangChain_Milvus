# 08_Animation_动画

## 学习目标
- 理解Vue的过渡动画系统
- 掌握基本的过渡动画使用方法
- 熟悉过渡组动画和列表动画
- 学会使用自定义过渡和动态过渡

## 知识点总结
1. **过渡动画**：使用transition组件实现元素的进入和离开动画
2. **过渡组动画**：使用transition-group组件实现列表的动画
3. **自定义过渡类名**：使用自定义的CSS类名控制动画
4. **动态过渡**：根据条件动态切换过渡效果
5. **交错过渡**：实现元素的交错过渡效果
6. **CSS动画**：使用CSS动画实现更复杂的动画效果
7. **列表动画**：实现列表的排序、添加和删除动画
