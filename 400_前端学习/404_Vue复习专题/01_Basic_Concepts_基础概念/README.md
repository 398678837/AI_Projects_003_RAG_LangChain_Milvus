# 01_Basic_Concepts_基础概念

## 学习目标
- 理解Vue的基本概念
- 掌握Vue的模板语法
- 熟悉Vue的响应式数据
- 学会使用Vue的指令和计算属性

## 知识点总结
1. **插值表达式**：使用{{}}在模板中显示数据
2. **指令**：v-bind、v-if、v-for、v-on等指令
3. **计算属性**：基于响应式数据计算派生值
4. **监听属性**：监听数据变化并执行操作
5. **方法**：定义组件的方法
6. **过滤器**：格式化数据显示
7. **修饰符**：修饰指令的行为
8. **组件**：创建和使用组件
9. **生命周期钩子**：组件生命周期的各个阶段
10. **响应式数据**：Vue的响应式系统
