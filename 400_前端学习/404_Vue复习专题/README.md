# 404_Vue复习专题

## 学习目标
- 复习Vue的核心概念和语法
- 掌握Vue的组件化开发
- 熟悉Vue的状态管理和路由
- 学会Vue的表单处理和动画
- 了解Vue的实战应用

## 目录结构
- **01_Basic_Concepts_基础概念**：Vue的基本概念和模板语法
- **02_Core_Syntax_核心语法**：Vue的核心语法和指令
- **03_Components_组件**：Vue的组件化开发
- **04_Lifecycle_生命周期**：Vue的生命周期钩子
- **05_State_Management_状态管理**：Vue的状态管理方式
- **06_Routing_路由**：Vue的路由配置和导航
- **07_Form_Handling_表单处理**：Vue的表单处理和验证
- **08_Animation_动画**：Vue的过渡和动画
- **09_Practical_Applications_实战应用**：Vue的实战项目

## 知识点总结
1. **Vue基础**：模板语法、指令、计算属性、监听属性
2. **组件化开发**：组件创建、传参、事件、插槽
3. **生命周期**：组件的生命周期钩子和应用场景
4. **状态管理**：组件内部状态、组件间通信、Vuex
5. **路由**：路由配置、导航、参数、守卫
6. **表单处理**：表单绑定、验证、修饰符
7. **动画**：过渡动画、列表动画、CSS动画
8. **实战应用**：待办事项、计数器、天气应用、购物车
