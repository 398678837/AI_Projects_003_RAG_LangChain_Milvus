# 07_Routing_路由

## 学习目标
- 掌握小程序的路由系统
- 理解页面栈的概念
- 学会各种路由方式的使用
- 掌握路由参数的传递

## 知识点总结
1. **路由方式**：
   - wx.navigateTo
   - wx.redirectTo
   - wx.switchTab
   - wx.reLaunch
   - wx.navigateBack
2. **页面栈**：
   - getCurrentPages
   - 页面栈限制
3. **参数传递**：
   - URL参数
   - 页面间通信
4. **TabBar**：
   - 配置
   - 切换
