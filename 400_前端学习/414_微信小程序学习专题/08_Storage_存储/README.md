# 08_Storage_存储

## 学习目标
- 掌握小程序的存储API
- 理解同步和异步存储的区别
- 学会存储的最佳实践
- 掌握存储容量限制

## 知识点总结
1. **存储API**：
   - wx.setStorageSync
   - wx.getStorageSync
   - wx.removeStorageSync
2. **异步存储**：
   - wx.setStorage
   - wx.getStorage
   - wx.removeStorage
3. **存储信息**：
   - wx.getStorageInfo
   - 容量限制
4. **最佳实践**：
   - 数据结构设计
   - 缓存策略
   - 清理策略
