# 04_API_API

## 学习目标
- 掌握小程序的常用API
- 理解API的使用场景
- 学会异步API的处理
- 掌握API的最佳实践

## 知识点总结
1. **界面API**：
   - wx.showToast
   - wx.showModal
   - wx.showLoading
   - wx.navigateTo
2. **媒体API**：
   - wx.chooseImage
   - wx.uploadFile
   - wx.downloadFile
3. **位置API**：
   - wx.getLocation
   - wx.chooseLocation
4. **设备API**：
   - wx.getSystemInfo
   - wx.vibrateShort
