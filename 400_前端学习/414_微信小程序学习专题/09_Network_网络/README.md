# 09_Network_网络

## 学习目标
- 掌握小程序的网络请求
- 理解请求的封装
- 学会上传下载
- 掌握网络状态监听

## 知识点总结
1. **网络请求**：
   - wx.request
   - GET/POST请求
   - 请求头配置
2. **请求封装**：
   - Promise封装
   - 拦截器
   - 错误处理
3. **上传下载**：
   - wx.uploadFile
   - wx.downloadFile
4. **网络状态**：
   - wx.getNetworkType
   - wx.onNetworkStatusChange
