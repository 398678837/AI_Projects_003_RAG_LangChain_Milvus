# 05_Page_Lifecycle_页面生命周期

## 学习目标
- 理解小程序的页面生命周期
- 掌握页面生命周期的触发时机
- 学会在合适的生命周期中处理逻辑
- 理解应用生命周期和页面生命周期的关系

## 知识点总结
1. **应用生命周期**：
   - onLaunch
   - onShow
   - onHide
   - onError
2. **页面生命周期**：
   - onLoad
   - onShow
   - onReady
   - onHide
   - onUnload
3. **页面路由**：
   - 页面栈
   - 路由方式
   - 生命周期触发顺序
