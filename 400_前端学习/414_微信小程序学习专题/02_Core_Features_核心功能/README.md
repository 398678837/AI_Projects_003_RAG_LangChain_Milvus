# 02_Core_Features_核心功能

## 学习目标
- 掌握小程序的核心功能
- 理解小程序的组件系统
- 学会使用小程序API
- 掌握小程序的事件处理

## 知识点总结
1. **核心功能**：
   - 页面渲染
   - 数据绑定
   - 事件系统
   - 组件化
2. **组件系统**：
   - 基础组件
   - 表单组件
   - 导航组件
   - 媒体组件
3. **事件处理**：
   - 事件绑定
   - 事件对象
   - 事件冒泡
   - 事件捕获
4. **数据处理**：
   - setData
   - 数据响应式
   - 双向绑定
