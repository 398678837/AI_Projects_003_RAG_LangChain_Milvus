# 03_Components_组件

## 学习目标
- 掌握小程序的组件系统
- 理解常用组件的使用
- 学会自定义组件
- 掌握组件的通信

## 知识点总结
1. **基础组件**：
   - view/text/image
   - button/input
   - scroll-view/swiper
2. **表单组件**：
   - form/input/textarea
   - checkbox/radio/switch
   - slider/picker
3. **导航组件**：
   - navigator
   - tabBar
4. **自定义组件**：
   - 组件定义
   - 组件通信
   - 组件生命周期
