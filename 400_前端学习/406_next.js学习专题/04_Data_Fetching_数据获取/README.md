# 04_Data_Fetching_数据获取

## 学习目标
- 理解Next.js的数据获取方法
- 掌握getStaticProps和getServerSideProps
- 熟悉getStaticPaths和动态路由
- 学会客户端数据获取和SWR库的使用

## 知识点总结
1. **getStaticProps**：静态生成时获取数据
2. **getServerSideProps**：服务端渲染时获取数据
3. **getStaticPaths**：为动态路由生成静态路径
4. **客户端数据获取**：使用useEffect进行数据获取
5. **SWR**：React数据获取库
