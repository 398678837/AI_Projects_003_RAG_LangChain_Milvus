# 01_Basic_Concepts_基础概念

- 理解Next.js的基本概念和工作原理

1. **页面组件**：Next.js的页面组件结构
2. **布局组件**：可复用的布局组件
3. **组件**：React组件的创建和使用
4. **样式**：CSS模块的使用
5. **环境变量**：Next.js的环境变量配置
6. **图片优化**：Next.js的图片组件
7. **字体优化**：Next.js的字体优化
