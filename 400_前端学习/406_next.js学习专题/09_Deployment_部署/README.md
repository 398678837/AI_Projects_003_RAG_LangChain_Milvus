# 09_Deployment_部署

## 学习目标
- 理解Next.js应用的部署流程
- 掌握环境变量配置和构建优化
- 熟悉Docker和容器化部署
- 学会使用Vercel和CI/CD

## 知识点总结
1. **环境变量配置**：使用.env.local文件管理环境变量
2. **构建命令**：使用npm run build构建项目
3. **Docker配置**：使用Dockerfile和docker-compose
4. **Vercel部署**：使用Vercel平台部署
5. **环境变量验证**：验证必需的环境变量
6. **构建优化**：配置next.config.js进行优化
7. **部署脚本**：创建自动化部署脚本
8. **CI/CD配置**：使用GitHub Actions
9. **健康检查**：实现健康检查端点
