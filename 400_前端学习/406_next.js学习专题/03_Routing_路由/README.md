# 03_Routing_路由

## 学习目标
- 理解Next.js的路由系统
- 掌握Link组件和useRouter钩子
- 熟悉编程式导航和路由事件
- 学会使用浅路由和路由预加载

## 知识点总结
1. **Link组件**：用于客户端导航
2. **useRouter钩子**：访问和操作路由状态
3. **编程式导航**：使用router.push()和router.replace()
4. **路由事件**：监听路由变化事件
5. **浅路由**：不重新加载页面的路由更新
6. **路由预加载**：提前加载页面内容
