# 02_Pages_页面

## 学习目标
- 理解Next.js的页面结构
- 掌握动态路由和嵌套路由
- 熟悉404页面和错误页面的创建
- 学会使用参数化路由

## 知识点总结
1. **基本页面**：静态页面的创建
2. **带参数的页面**：使用动态路由参数
3. **动态路由页面**：使用catch-all路由
4. **可选的动态路由页面**：使用可选的catch-all路由
5. **嵌套路由**：创建嵌套的页面结构
6. **404页面**：自定义404页面
7. **错误页面**：自定义错误页面
