# 03_UI_Development_UI开发

## 学习目标
- 掌握XML布局文件的使用
- 理解常见的布局容器
- 学会使用常用控件
- 掌握样式和主题的使用
- 理解Material Design设计规范

## 关键要点
### 1. 布局容器
- LinearLayout：线性布局，水平或垂直排列
- RelativeLayout：相对布局，相对位置排列
- ConstraintLayout：约束布局，灵活的约束关系
- FrameLayout：帧布局，层叠排列
- TableLayout：表格布局，行列排列

### 2. 常用控件
- TextView：显示文本
- EditText：输入文本
- Button：按钮
- ImageView：显示图片
- CheckBox：复选框
- RadioButton：单选按钮
- Switch：开关
- ProgressBar：进度条
- RecyclerView：列表（替代ListView）
- ViewPager：页面滑动

### 3. 样式和主题
- Style：一组属性的集合，可应用于控件
- Theme：应用于Activity或整个应用的样式
- 继承：使用parent属性继承样式
- 自定义属性：在attrs.xml中定义

### 4. Material Design
- Material Components：MD设计组件库
- Color Palette：颜色调色板
- Typography：字体排版
- Elevation：阴影和高度
- Animation：动画效果

### 5. 资源文件
- res/layout：布局文件
- res/values：字符串、颜色、尺寸、样式
- res/drawable：图片、形状
- res/mipmap：应用图标
