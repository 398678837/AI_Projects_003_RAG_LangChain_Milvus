# 05_Classes_类

## 学习目标
- 理解TypeScript类的概念和用途
- 掌握类的基本语法和特性
- 熟悉类的继承和多态
- 学会使用类的访问修饰符和特殊方法

## 知识点总结
1. **基本类**：定义类的基本结构和方法
2. **访问修饰符**：public、private、protected
3. **继承**：子类继承父类的属性和方法
4. **只读属性**：使用readonly标记只读属性
5. **静态属性和方法**：使用static定义静态成员
6. **抽象类**：定义抽象方法和属性
7. **接口实现**：类实现接口的方法
8. **构造函数简写**：简化构造函数的写法
9. **Getter 和 Setter**：访问和修改私有属性
10. **类表达式**：使用表达式定义类
11. **泛型类**：使用泛型参数化类
12. **类的类型**：类作为类型使用
13. **类的实例类型**：类的实例作为类型使用
14. **类的继承与多态**：多态的实现
15. **类的私有方法**：只能在类内部访问的方法
16. **类的静态块**：静态初始化代码
17. **类的装饰器**：使用装饰器增强类的功能
18. **类的属性装饰器**：装饰类的属性
19. **类的方法装饰器**：装饰类的方法
20. **类的参数装饰器**：装饰方法的参数
