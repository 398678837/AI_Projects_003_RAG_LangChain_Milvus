# 06_Modules_模块化

## 学习目标
- 理解TypeScript模块的概念和用途
- 掌握模块的导出和导入语法
- 熟悉模块的解析策略
- 学会使用模块组织代码

## 知识点总结
1. **导出声明**：使用export关键字导出变量、函数和类
2. **导入声明**：使用import关键字导入模块
3. **重命名导入**：使用as关键字重命名导入
4. **导入所有**：使用* as导入模块的所有导出
5. **默认导出**：使用export default导出默认值
6. **混合导出**：同时使用默认导出和命名导出
7. **重新导出**：从一个模块导出另一个模块的内容
8. **模块解析**：TypeScript如何查找模块
9. **环境模块**：声明外部模块
10. **模块增强**：增强现有模块的类型
11. **动态导入**：使用import()函数动态导入模块
12. **条件导入**：根据条件导入模块
13. **命名空间**：使用命名空间组织代码
14. **模块与命名空间的结合**：结合使用模块和命名空间
15. **循环依赖**：处理模块之间的循环依赖
16. **模块解析策略**：配置模块解析策略
17. **模块类型**：CommonJS和ES模块
18. **CommonJS 模块**：使用require和module.exports
19. **ES 模块**：使用import和export
20. **模块打包**：使用webpack等工具打包模块
