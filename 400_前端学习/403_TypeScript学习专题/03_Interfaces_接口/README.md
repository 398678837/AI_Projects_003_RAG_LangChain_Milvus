# 03_Interfaces_接口

## 学习目标
- 理解TypeScript接口的概念和用途
- 掌握接口的基本语法和特性
- 熟悉接口的继承和合并
- 学会使用接口定义对象、函数和类的类型

## 知识点总结
1. **基本接口**：定义对象的结构
2. **可选属性**：使用?标记可选属性
3. **只读属性**：使用readonly标记只读属性
4. **索引签名**：定义对象的索引类型
5. **函数类型**：定义函数的参数和返回值类型
6. **可索引类型**：定义数组和对象的索引访问
7. **类类型**：定义类必须实现的方法和属性
8. **接口继承**：一个接口继承另一个接口的属性和方法
9. **多重继承**：一个接口可以继承多个接口
10. **接口合并**：多个同名接口会自动合并
11. **接口与类型别名的区别**：接口可以合并，类型别名不能
12. **接口作为函数参数**：使用接口定义函数参数的类型
13. **接口作为函数返回值**：使用接口定义函数返回值的类型
14. **接口与泛型**：使用泛型参数化接口
15. **接口与联合类型**：使用接口定义联合类型的成员
16. **接口与交叉类型**：使用接口定义交叉类型的成员
17. **接口与类型保护**：使用接口进行类型保护
18. **接口与可选链**：安全地访问接口的可选属性
19. **接口与空值合并**：为接口的可选属性提供默认值
20. **接口与断言**：使用断言将其他类型转换为接口类型
