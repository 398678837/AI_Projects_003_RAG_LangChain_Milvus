# 07_Advanced_Types_高级类型

## 学习目标
- 理解TypeScript的高级类型
- 掌握交叉类型和联合类型
- 熟悉类型守卫和类型谓词
- 学会使用映射类型和条件类型

## 知识点总结
1. **交叉类型**：使用&组合多个类型
2. **联合类型**：使用|表示多种类型之一
3. **类型守卫**：检查值的类型
4. **类型谓词**：自定义类型守卫
5. **类型保护**：在运行时检查类型
6. **类型别名**：为类型创建别名
7. **字面量类型**：使用特定的值作为类型
8. **映射类型**：基于现有类型创建新类型
9. **条件类型**：根据条件创建类型
10. **分布式条件类型**：对联合类型的每个成员应用条件
11. **类型推断**：使用infer关键字推断类型
12. **索引类型**：使用keyof和索引访问类型
13. **模板字面量类型**：使用模板字面量创建类型
14. **递归类型**：类型引用自身
15. **条件类型与映射类型结合**：创建复杂的类型转换
16. **类型参数约束**：限制类型参数的范围
17. **类型参数默认值**：为类型参数提供默认值
18. **类型断言函数**：使用asserts关键字
19. **可调用类型**：定义可调用的对象类型
20. **构造函数类型**：定义构造函数的类型
