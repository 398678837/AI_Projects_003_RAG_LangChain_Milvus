# 02_Type_System_类型系统

## 学习目标
- 理解TypeScript的类型系统
- 掌握基本类型和复合类型
- 熟悉类型断言和类型守卫
- 学会使用泛型和类型约束

## 知识点总结
1. **基本类型**：string、number、boolean、undefined、null、symbol、bigint
2. **数组类型**：number[] 或 Array<number>
3. **元组类型**：固定长度和类型的数组
4. **枚举类型**：命名常量集合
5. **字面量类型**：使用特定的值作为类型
6. **联合类型**：允许值为多种类型之一
7. **交叉类型**：组合多个类型
8. **类型别名**：为类型创建别名
9. **类型断言**：告诉TypeScript编译器某个值的类型
10. **类型守卫**：检查值的类型
11. **类型保护**：在运行时检查类型
12. **类型推断**：TypeScript自动推断类型
13. **类型兼容性**：子类型可以赋值给父类型
14. **类型收窄**：通过条件语句收窄类型
15. **类型谓词**：自定义类型守卫
16. **索引签名**：定义对象的索引类型
17. **函数重载**：定义多个函数签名
18. **泛型函数**：使用类型参数
19. **泛型约束**：限制类型参数
20. **类型参数推断**：TypeScript自动推断类型参数
