# 08_Declaration_Files_声明文件

## 学习目标
- 理解TypeScript声明文件的概念和用途
- 掌握声明文件的基本语法
- 熟悉声明全局变量、函数和类型
- 学会创建和使用声明文件

## 知识点总结
1. **声明全局变量**：使用declare var声明全局变量
2. **声明全局函数**：使用declare function声明全局函数
3. **声明全局类型**：使用declare interface声明全局接口
4. **声明模块**：使用declare module声明模块
5. **声明命名空间**：使用declare namespace声明命名空间
6. **声明类**：使用declare class声明类
7. **声明枚举**：使用declare enum声明枚举
8. **声明接口**：使用declare interface声明接口
9. **声明类型别名**：使用declare type声明类型别名
10. **声明模块增强**：增强现有模块的类型
11. **声明环境模块**：声明非TypeScript模块
12. **声明合并**：多个声明会自动合并
13. **声明泛型**：使用泛型参数
14. **声明函数重载**：声明多个函数签名
15. **声明可选参数和默认参数**：使用?和默认值
16. **声明剩余参数**：使用...语法
17. **声明异步函数**：使用async和Promise
18. **声明生成器函数**：使用function*语法
19. **声明装饰器**：声明装饰器函数
20. **声明模块导出**：声明模块的导出内容
