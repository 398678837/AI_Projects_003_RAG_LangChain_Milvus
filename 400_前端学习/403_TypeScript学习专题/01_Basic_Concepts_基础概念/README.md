# 01_Basic_Concepts_基础概念

## 学习目标
- 理解TypeScript的基本概念
- 掌握TypeScript的类型注解
- 熟悉TypeScript的变量声明和函数类型
- 学会使用TypeScript的基本类型

## 知识点总结
1. **类型注解**：使用类型注解指定变量和函数的类型
2. **变量声明**：使用let、const和var声明变量
3. **函数类型**：指定函数参数和返回值的类型
4. **数组类型**：定义数组的类型
5. **元组类型**：定义固定长度和类型的数组
6. **枚举类型**：定义命名常量集合
7. **任意类型**：使用any类型允许任何值
8. **空类型**：使用void表示函数无返回值
9. **类型断言**：告诉TypeScript编译器某个值的类型
10. **接口**：定义对象的结构
11. **类**：使用类定义对象
12. **泛型**：使用泛型实现类型参数化
13. **类型守卫**：检查值的类型
14. **联合类型**：允许值为多种类型之一
15. **交叉类型**：组合多个类型
16. **类型别名**：为类型创建别名
17. **字面量类型**：使用特定的值作为类型
18. **非空断言**：告诉TypeScript编译器值不为null或undefined
19. **可选链**：安全地访问可能不存在的属性
20. **空值合并运算符**：提供默认值
