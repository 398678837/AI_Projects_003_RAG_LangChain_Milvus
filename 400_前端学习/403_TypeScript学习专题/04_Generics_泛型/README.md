# 04_Generics_泛型

## 学习目标
- 理解TypeScript泛型的概念和用途
- 掌握泛型函数和泛型类的使用
- 熟悉泛型约束和泛型默认类型
- 学会使用泛型实现类型安全的代码

## 知识点总结
1. **泛型函数**：使用类型参数创建可重用的函数
2. **泛型函数类型推断**：TypeScript自动推断类型参数
3. **多个泛型参数**：使用多个类型参数
4. **泛型约束**：限制类型参数的范围
5. **泛型类**：使用类型参数创建可重用的类
6. **泛型接口**：使用类型参数创建可重用的接口
7. **泛型类型别名**：使用类型参数创建可重用的类型别名
8. **泛型默认类型**：为类型参数提供默认值
9. **泛型与映射类型**：使用泛型创建映射类型
10. **泛型与条件类型**：使用泛型创建条件类型
11. **泛型与类型推断**：TypeScript自动推断泛型类型
12. **泛型与函数重载**：使用泛型实现函数重载
13. **泛型与联合类型**：使用泛型限制联合类型
14. **泛型与交叉类型**：使用泛型创建交叉类型
15. **泛型与索引类型**：使用泛型和索引类型访问对象属性
16. **泛型与递归类型**：使用泛型创建递归类型
17. **泛型与 Promise**：使用泛型处理异步操作
18. **泛型与工厂函数**：使用泛型创建工厂函数
19. **泛型与装饰器**：使用泛型实现装饰器
20. **泛型与模块**：在模块中使用泛型
