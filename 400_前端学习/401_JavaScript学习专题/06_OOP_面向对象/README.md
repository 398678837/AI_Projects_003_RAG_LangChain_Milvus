# 06_OOP_面向对象

## 学习目标
- 理解面向对象编程的基本概念
- 掌握ES6类的定义和使用
- 熟悉继承、封装和多态
- 学会使用面向对象编程思想解决问题

## 知识点总结
1. **构造函数**：传统的面向对象实现方式
2. **ES6类**：更简洁的类定义语法
3. **继承**：使用extends关键字
4. **封装**：使用getter和setter
5. **多态**：方法重写
6. **抽象类**：不能直接实例化的类
7. **接口**：定义行为规范
8. **混入**：使用Object.assign实现
