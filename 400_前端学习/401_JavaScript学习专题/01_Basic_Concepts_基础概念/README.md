# 01_Basic_Concepts_基础概念

## 学习目标
- 理解JavaScript的基本概念
- 掌握变量声明和数据类型
- 熟悉运算符和控制流
- 学会定义和调用函数

## 知识点总结
1. **变量声明**：使用let、const和var
2. **数据类型**：number、string、boolean、null、undefined、object、array
3. **运算符**：算术运算符、比较运算符、逻辑运算符
4. **控制流**：if-else、switch、for、while
5. **函数**：函数定义、参数、返回值
