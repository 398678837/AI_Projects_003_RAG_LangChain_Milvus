# 05_ES6_Plus

## 学习目标
- 掌握ES6及以上版本的新特性
- 理解箭头函数、模板字符串和块级作用域
- 熟悉解构赋值、扩展运算符和类
- 学会使用Set、Map、可选链和空值合并等新特性

## 知识点总结
1. **块级作用域**：let和const关键字
2. **箭头函数**：简化函数定义
3. **模板字符串**：更灵活的字符串拼接
4. **解构赋值**：快速提取对象和数组中的值
5. **扩展运算符**：合并数组和对象
6. **类**：面向对象编程的新语法
7. **模块**：import和export
8. **迭代器和生成器**：function*
9. **Set和Map**：新的数据结构
10. **可选链和空值合并**：?.和??操作符
