# 03_DOM_操作

## 学习目标
- 掌握DOM元素的选择方法
- 理解DOM元素的创建、修改和删除
- 熟悉DOM事件处理机制
- 学会使用DOM操作实现页面交互

## 知识点总结
1. **元素选择**：getElementById、getElementsByClassName、getElementsByTagName、querySelector、querySelectorAll
2. **内容修改**：textContent、innerHTML
3. **属性操作**：setAttribute、getAttribute、removeAttribute
4. **元素操作**：createElement、appendChild、insertBefore、removeChild
5. **样式修改**：style属性、classList
6. **事件处理**：addEventListener、removeEventListener、事件冒泡和捕获
