# 02_Core_Syntax_核心语法

## 学习目标
- 掌握ES6+核心语法特性
- 理解箭头函数和模板字符串
- 熟悉解构赋值和扩展运算符
- 学会使用默认参数和剩余参数

## 知识点总结
1. **箭头函数**：简化函数定义
2. **模板字符串**：更灵活的字符串拼接
3. **解构赋值**：快速提取对象和数组中的值
4. **扩展运算符**：合并数组和对象
5. **默认参数**：为函数参数设置默认值
6. **剩余参数**：处理可变数量的函数参数
