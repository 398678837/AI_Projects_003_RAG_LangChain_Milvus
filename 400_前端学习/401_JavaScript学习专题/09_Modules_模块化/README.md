# 09_Modules_模块化

## 学习目标
- 理解模块化的概念和重要性
- 掌握ES6模块的导入和导出
- 熟悉CommonJS、AMD和UMD模块系统
- 学会使用模块化组织代码

## 知识点总结
1. **ES6 模块**：
   - 导出：export、export default
   - 导入：import、import * as
2. **CommonJS 模块**：
   - 导出：module.exports
   - 导入：require()
3. **AMD 模块**：
   - 定义：define()
   - 加载：require()
4. **UMD 模块**：通用模块定义，兼容多种模块系统
5. **模块的好处**：
   - 代码组织
   - 命名空间
   - 依赖管理
   - 代码复用
   - 维护性
