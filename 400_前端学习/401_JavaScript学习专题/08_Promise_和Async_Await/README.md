# 08_Promise_和Async_Await

## 学习目标
- 理解Promise的概念和工作原理
- 掌握Promise的各种方法和使用场景
- 熟悉Async/Await语法和错误处理
- 学会使用Promise和Async/Await处理异步操作

## 知识点总结
1. **Promise基础**：创建、resolve、reject、then、catch、finally
2. **Promise链式调用**：处理多个异步操作
3. **Promise方法**：
   - Promise.all：并行执行多个Promise
   - Promise.race：返回第一个完成的Promise
   - Promise.allSettled：等待所有Promise完成
   - Promise.any：返回第一个成功的Promise
4. **Async/Await**：基于Promise的语法糖
5. **错误处理**：try/catch/finally
