# 04_Asynchronous_异步编程

## 学习目标
- 理解异步编程的概念和重要性
- 掌握回调函数、Promise和Async/Await
- 熟悉Promise的各种方法和使用场景
- 学会处理异步错误和并发操作

## 知识点总结
1. **回调函数**：处理异步操作的传统方式
2. **Promise**：更优雅的异步处理方式
3. **Async/Await**：基于Promise的语法糖
4. **Promise方法**：Promise.all、Promise.race、Promise.resolve、Promise.reject
5. **错误处理**：try/catch、Promise.catch
