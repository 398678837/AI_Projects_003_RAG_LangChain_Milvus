# 基础概念与环境

## 1. C++语言简介

C++是一种通用的、高级的编程语言，是C语言的扩展和改进版本。它融合了面向过程编程、面向对象编程和泛型编程的特性，被广泛应用于系统软件、游戏开发、嵌入式系统、金融分析等领域。

### 1.1 C++的历史

- **1979年**：Bjarne Stroustrup开始开发C++，最初称为"C with Classes"
- **1983年**：正式更名为C++
- **1985年**：发布第一版C++编译器和《The C++ Programming Language》书籍
- **1998年**：ISO C++标准正式发布（C++98）
- **2003年**：C++03标准发布，对C++98进行了小幅修正
- **2011年**：C++11标准发布，引入了许多新特性
- **2014年**：C++14标准发布
- **2017年**：C++17标准发布
- **2020年**：C++20标准发布
- **2023年**：C++23标准发布

### 1.2 C++的特点

- **兼容性**：与C语言兼容，可以直接使用C语言代码
- **面向对象**：支持类、继承、多态等面向对象特性
- **泛型编程**：支持模板和STL标准库
- **高效性**：接近底层，执行效率高
- **灵活性**：支持多种编程范式
- **丰富的标准库**：提供了大量的工具和组件

## 2. 开发环境搭建

### 2.1 编译器选择

- **GCC**：开源编译器，支持多种平台
- **Clang**：基于LLVM的编译器，错误信息友好
- **MSVC**：微软的C++编译器，与Visual Studio集成
- **Intel C++ Compiler**：针对Intel处理器优化的编译器

### 2.2 集成开发环境（IDE）

- **Visual Studio**：功能强大的IDE，适合Windows平台
- **Visual Studio Code**：轻量级编辑器，支持多种插件
- **CLion**：JetBrains开发的C++ IDE，功能丰富
- **Code::Blocks**：开源IDE，跨平台
- **Xcode**：Apple开发的IDE，适合macOS平台

### 2.3 环境配置

#### Windows平台

1. 安装Visual Studio或MinGW
2. 配置环境变量
3. 验证安装：`g++ --version`

#### macOS平台

1. 安装Xcode Command Line Tools：`xcode-select --install`
2. 或安装Homebrew后安装GCC：`brew install gcc`
3. 验证安装：`g++ --version`

#### Linux平台

1. 安装GCC：`sudo apt-get install g++`（Ubuntu/Debian）或 `sudo yum install gcc-c++`（CentOS/RHEL）
2. 验证安装：`g++ --version`

## 3. 第一个C++程序

### 3.1 程序结构

一个基本的C++程序包括以下部分：

- **预处理指令**：以`#`开头的指令，如`#include`
- **函数**：程序的执行单元，如`main()`函数
- **语句**：执行具体操作的代码行
- **注释**：解释代码的文本，不被编译器执行

### 3.2 Hello World示例

```cpp
#include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}
```

### 3.3 编译和运行

#### 使用命令行编译

```bash
# 编译
g++ main.cpp -o main

# 运行
./main  # Linux/macOS
main.exe  # Windows
```

#### 使用IDE编译

1. 创建新项目
2. 添加源文件
3. 构建项目
4. 运行程序

## 4. 基本数据类型

### 4.1 内置数据类型

| 类型 | 大小 | 描述 |
|------|------|------|
| `bool` | 1字节 | 布尔值（true/false） |
| `char` | 1字节 | 字符 |
| `int` | 4字节 | 整数 |
| `float` | 4字节 | 单精度浮点数 |
| `double` | 8字节 | 双精度浮点数 |
| `void` | 无 | 无类型 |

### 4.2 类型修饰符

- `signed`：有符号类型
- `unsigned`：无符号类型
- `short`：短类型
- `long`：长类型
- `long long`：更长的类型

### 4.3 类型转换

- **隐式转换**：编译器自动进行的转换
- **显式转换**：使用类型转换运算符进行的转换

```cpp
int a = 10;
double b = a;  // 隐式转换
int c = static_cast<int>(b);  // 显式转换
```

## 5. 变量和常量

### 5.1 变量

变量是存储数据的容器，需要先声明后使用。

```cpp
// 声明变量
int age;
double salary;
char grade;

// 声明并初始化
int count = 0;
double pi = 3.14159;
char initial = 'A';
```

### 5.2 常量

常量是值不能改变的变量，使用`const`关键字声明。

```cpp
const int MAX_SIZE = 100;
const double PI = 3.14159;
const char NEWLINE = '\n';
```

### 5.3 变量作用域

- **局部变量**：在函数或代码块内声明，只在该范围内有效
- **全局变量**：在函数外声明，在整个程序中有效
- **静态变量**：使用`static`关键字声明，生命周期贯穿整个程序

## 7. 输入输出

### 7.1 标准输入输出

C++使用`iostream`库进行输入输出操作：

- `cout`：标准输出流
- `cin`：标准输入流
- `endl`：换行符

```cpp
#include <iostream>

int main() {
    int age;
    std::cout << "请输入你的年龄：";
    std::cin >> age;
    std::cout << "你的年龄是：" << age << std::endl;
    return 0;
}
```

### 7.2 格式化输出

可以使用`iomanip`库进行格式化输出：

```cpp
#include <iostream>
#include <iomanip>

int main() {
    double pi = 3.1415926535;
    std::cout << std::fixed << std::setprecision(2);
    std::cout << "PI = " << pi << std::endl;
    return 0;
}
```

## 8. 注释

### 8.1 单行注释

使用`//`开头的注释，只在一行内有效：

```cpp
// 这是一个单行注释
int x = 10;  // 这也是一个单行注释
```

### 8.2 多行注释

使用`/* */`包围的注释，可以跨越多行：

```cpp
/*
这是一个多行注释
可以跨越多行
*/
int y = 20;
```

## 9. 编码规范

### 9.1 命名规范

- **变量名**：使用小写字母和下划线，如`user_name`
- **常量名**：使用大写字母和下划线，如`MAX_VALUE`
- **函数名**：使用驼峰命名法，如`calculateTotal()`
- **类名**：使用驼峰命名法，首字母大写，如`Person`

### 9.2 代码风格

- 使用4个空格进行缩进
- 左括号放在行尾，右括号放在单独的一行
- 每行只写一条语句
- 适当使用空行分隔代码块

## 10. 常见问题与解决方案

### 10.1 编译错误

- **未定义的引用**：检查是否缺少头文件或函数定义
- **语法错误**：检查代码的语法是否正确
- **类型不匹配**：检查变量类型是否一致

### 10.2 运行时错误

- **段错误**：检查是否访问了无效的内存地址
- **逻辑错误**：检查程序的逻辑是否正确
- **内存泄漏**：检查是否正确释放了动态分配的内存
