#### 1.2 KeyGenMe分析
- **算法分析**：分析注册码生成算法
- **注册码生成**：理解算法后生成有效的注册码
- **KeyGen编写**：编写注册机程序
- **示例**：
  ```python
  # KeyGen示例
  def generate_key(name):
      # 简单的注册码生成算法
      key = []
      for i, c in enumerate(name):
          key.append(str(ord(c) ^ (i + 1)))
      return '-'.join(key)

  # 测试
  name = 'test'
  key = generate_key(name)
  print(f"Name: {name}, Key: {key}")
  ```
