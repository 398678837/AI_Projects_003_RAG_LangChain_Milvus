### 1. 安全分析环境

#### 1.1 虚拟机
- **VMware**：功能强大，支持快照和克隆
- **VirtualBox**：开源免费，支持多种操作系统
- **Hyper-V**：Windows内置，性能较好

#### 1.2 网络隔离
- **仅主机模式**：与主机网络隔离，安全可靠

- **断开网络**：完全隔离，适合分析恶意代码
- **建议**：分析恶意代码时使用断开网络或仅主机模式

- **快速恢复**：遇到问题时快速恢复到干净状态

### 2. 分析流程

#### 2.1 信息收集
- **文件Hash**：计算MD5, SHA1, SHA256哈希值
- **文件类型**：识别文件类型和结构
- **基本信息**：文件名, 文件大小, 创建时间, 修改时间
- **示例**：
  ```python
  import hashlib
  import os

  def collect_info(file_path):
      # 计算哈希值
      md5 = hashlib.md5()
      sha1 = hashlib.sha1()
      sha256 = hashlib.sha256()

      with open(file_path, 'rb') as f:
          while True:
              data = f.read(4096)
              if not data:
                  break
              md5.update(data)
              sha1.update(data)
              sha256.update(data)

      # 获取文件信息
      file_info = {}
      file_info['file_path'] = file_path
      file_info['file_name'] = os.path.basename(file_path)
      file_info['file_size'] = os.path.getsize(file_path)
      file_info['md5'] = md5.hexdigest()
      file_info['sha1'] = sha1.hexdigest()
      file_info['sha256'] = sha256.hexdigest()

      return file_info

  # 测试
  file_path = 'sample.exe'
  info = collect_info(file_path)
  for key, value in info.items():
      print(f'{key}: {value}')
  ```

      # 静态分析
      print(f'Analyzing sample: {sample_path}')
      print(f'MD5: {md5}')
      print(f'SHA1: {sha1}')

      # 动态分析
      print('Running dynamic analysis...')
