# SciPy线性代数

本示例展示了SciPy的线性代数功能，包括矩阵运算、特征值计算、线性方程组求解等。

## 功能说明

- 基本线性代数操作：矩阵加法、减法、乘法等
- 矩阵属性：行列式、迹、秩等
- 矩阵逆和伪逆：计算矩阵的逆和伪逆
- 特征值和特征向量：计算矩阵的特征值和特征向量
- 奇异值分解：对矩阵进行奇异值分解
- 求解线性方程组：求解Ax = b类型的线性方程组
- 最小二乘问题：求解超定方程组的最小二乘解
- 矩阵分解：LU分解、Cholesky分解、QR分解
- 范数计算：计算矩阵的各种范数
- 条件数：计算矩阵的条件数
