# LangGraph实战应用

本示例展示了如何使用LangGraph构建一个完整的流程应用，包含意图识别、知识库查询、对话处理等功能。

## 功能说明

- 实现完整的用户请求处理流程
- 包含4个节点：意图识别、知识库查询、对话处理、其他处理
- 使用条件边根据意图选择不同的处理路径
- 集成模拟知识库
- 适配Qwen等通用大模型
- 支持批量测试和交互式模式

## 代码结构

```python
# 1. 配置API密钥和大模型
# 2. 实现模拟知识库查询函数
# 3. 定义状态类型
# 4. 定义节点函数
# 5. 定义条件边函数
# 6. 创建状态图
# 7. 测试流程
```

## 配置说明

1. **API密钥配置**：
   - 在环境变量中设置 `API_KEY`
   - 或直接在代码中修改 `api_key` 变量

2. **大模型配置**：
   - 默认使用Qwen模型的API地址
   - 可以根据需要修改 `base_url`

## 运行示例

1. 安装依赖：
   ```bash
   pip install -r ../../requirements.txt
   ```

2. 运行示例：
   ```bash
   python demo.py
   ```

3. 交互模式：
   - 输入内容，按回车获取回答
   - 输入'quit'退出

## 学习要点

- **完整流程设计**：如何设计一个完整的用户请求处理流程
- **意图识别**：如何识别用户的意图并选择相应的处理路径
- **知识库集成**：如何集成知识库查询功能
- **模块化设计**：如何将复杂任务分解为多个子任务
