# LangGraph状态管理

本示例展示了如何使用LangGraph管理和持久化流程状态，支持多轮对话和状态恢复。

## 功能说明

- 实现状态的序列化和反序列化
- 支持状态持久化到文件
- 实现多轮对话历史管理
- 适配Qwen等通用大模型
- 支持会话状态的保存和加载

## 代码结构

```python
# 1. 配置API密钥和大模型
# 2. 定义状态类型（包含序列化和反序列化方法）
# 3. 实现状态持久化和加载函数
# 4. 定义节点函数
# 5. 创建状态图
# 6. 测试流程
```

## 配置说明

1. **API密钥配置**：
   - 在环境变量中设置 `API_KEY`
   - 或直接在代码中修改 `api_key` 变量

2. **大模型配置**：
   - 默认使用Qwen模型的API地址
   - 可以根据需要修改 `base_url`

## 运行示例

1. 安装依赖：
   ```bash
   pip install -r ../../requirements.txt
   ```

2. 运行示例：
   ```bash
   python demo.py
   ```

## 学习要点

- **状态管理**：如何定义和管理流程状态
- **状态持久化**：如何将状态保存到文件并加载
- **多轮对话**：如何维护和使用对话历史
- **序列化**：如何实现状态的序列化和反序列化
