# 向量库基础

## 什么是向量库？

向量库是一种专门用于存储和检索向量数据的数据库。它可以高效地存储和检索大规模的向量数据，广泛应用于RAG系统、推荐系统、图像检索等领域。

## 核心概念

### 1. 向量 (Vector)
- 文字转成的数字数组（Embedding结果）
- 通常是高维向量，长度为几百到几千

### 2. 集合 (Collection)
- 相当于普通数据库的「表」
- 用于存储一组相关的向量数据

### 3. 入库
- 把切片后的文档向量存进库
- 支持批量插入和更新

### 4. 检索
- 输入问题向量，找出语义最相似的文本
- 使用近似最近邻（ANN）算法，如HNSW、IVF等

### 5. 相似度
- 用余弦相似度判断文本语义远近
- 相似度越高，语义越接近

## 工作流程

1. **文本切片**: 将长文本切分成短文本
2. **向量生成**: 将短文本转换为向量表示
3. **向量入库**: 将向量存储到向量库中
4. **语义检索**: 输入问题向量，找出语义最相似的文本
5. **结果返回**: 将检索结果返回给用户

## 应用场景

- **RAG系统**: 向量库用于存储知识库的向量表示，实现语义相似检索
- **推荐系统**: 向量库用于存储用户和物品的向量表示，实现个性化推荐
- **图像检索**: 向量库用于存储图像的向量表示，实现相似图像检索
- **语音识别**: 向量库用于存储语音的向量表示，实现语音识别和检索

## 总结

向量库是一种专门用于存储和检索向量数据的数据库，广泛应用于RAG系统、推荐系统、图像检索等领域。本目录提供了向量库的基本概念和技术，帮助您快速掌握向量库的基础知识。
