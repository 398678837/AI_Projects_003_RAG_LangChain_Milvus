# 向量相似度

## 什么是向量相似度？

向量相似度是指两个向量之间的相似程度，通常用余弦相似度、欧氏距离等指标来衡量。向量相似度可以用于语义检索、文本分类、聚类分析等任务。

## 常用的相似度计算方法

### 1. 余弦相似度
- 衡量两个向量之间的夹角余弦值
- 取值范围为[-1, 1]
- 值越接近1，说明两个向量越相似

### 2. 欧氏距离
- 衡量两个向量之间的直线距离
- 取值范围为[0, +∞)
- 值越小，说明两个向量越相似

### 3. 曼哈顿距离
- 衡量两个向量之间的曼哈顿距离
- 取值范围为[0, +∞)
- 值越小，说明两个向量越相似

## 相似度计算的步骤

1. **生成向量**: 将需要计算相似度的文本转换为向量表示
2. **选择相似度计算方法**: 选择合适的相似度计算方法，如余弦相似度、欧氏距离等
3. **计算相似度**: 使用选择的方法计算两个向量之间的相似度
4. **输出相似度**: 将计算结果输出，用于后续任务

## 相似度的应用

### 1. 语义检索
- 输入问题向量，找出语义最相似的文本
- 广泛应用于RAG系统、搜索引擎等领域

### 2. 文本分类
- 将向量输入分类模型，进行文本分类
- 广泛应用于情感分析、垃圾邮件检测等领域

### 3. 聚类分析
- 将向量输入聚类模型，进行聚类分析
- 广泛应用于主题建模、用户分群等领域

### 4. 推荐系统
- 将向量用于个性化推荐
- 广泛应用于电商、视频等领域
