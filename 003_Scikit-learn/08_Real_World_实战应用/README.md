- 分类实战：鸢尾花分类，展示完整的分类流程

- 预处理实战：处理混合类型数据，展示如何处理数值和分类特征
- 模型选择实战：网格搜索调优，展示如何优化模型参数
- 集成学习实战：堆叠模型，展示如何组合多个模型提高性能
