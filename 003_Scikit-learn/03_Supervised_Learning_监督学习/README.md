# Scikit-learn监督学习

本示例展示了Scikit-learn中各种监督学习算法，包括分类算法、回归算法、支持向量机、决策树和随机森林以及集成学习。

## 功能说明

- 分类算法：包括逻辑回归、K近邻、决策树、随机森林和支持向量机
- 回归算法：包括线性回归、岭回归、lasso回归、K近邻回归、决策树回归和随机森林回归
- 支持向量机：包括分类和回归
- 决策树和随机森林：包括分类和回归
- 集成学习：包括AdaBoost、Gradient Boosting和Voting Classifier

## 学习要点

- **分类算法**：用于预测离散标签，如逻辑回归、K近邻、决策树等
- **回归算法**：用于预测连续值，如线性回归、岭回归、lasso回归等
- **支持向量机**：可以用于分类和回归，通过寻找最优超平面来分离数据
- **决策树和随机森林**：基于树的方法，随机森林是决策树的集成
- **集成学习**：通过组合多个模型来提高预测性能，如AdaBoost、Gradient Boosting等
