# Scikit-learn模型评估

本示例展示了Scikit-learn中各种模型评估方法，包括分类模型评估、回归模型评估、交叉验证、模型选择和模型持久化。

## 功能说明

- 分类模型评估：包括准确率、精确率、召回率、F1-score、混淆矩阵和分类报告
- 回归模型评估：包括均方误差、均方根误差、平均绝对误差、R2分数等
- 交叉验证：包括K折交叉验证和分层K折交叉验证
- 模型选择：包括网格搜索和随机搜索
- 模型持久化：保存和加载模型

## 学习要点

- **分类模型评估**：使用各种指标评估分类模型的性能
- **回归模型评估**：使用各种指标评估回归模型的性能
